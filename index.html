1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI 모델(Anthropic Claude Fable 5/Mythos 5)의 오남용 방지를 위한 정부 차원의 수출 통제 및 접근 제한과, 이를 우회하려는 공격자들의 시도가 대립하는 양상입니다. 특히 Arch Linux AUR 패키지 오염을 통한 Infostealer 유포 및 eBPF Rootkit 설치 등 오픈소스 생태계를 겨냥한 공급망 공격이 심화되고 있습니다. 또한, AI 에이전트의 메모리 계층(LangGraph Checkpointer)을 악용한 RCE 공격과 같은 새로운 공격 표면이 등장하며 보안 담당자들의 대응 범위가 급격히 확장되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • ShinyHunters (UNC6240): Oracle PeopleSoft의 Zero-Day 취약점을 악용하여 미국 교육 기관을 대상으로 대규모 데이터 탈취 및 갈취 캠페인 수행.

  • 중국 연계 Smishing 네트워크: Google Gemini AI 에이전트를 악용하여 미국 시민 대상 피싱 문자 발송(Google 법적 대응 중).

  • The Gentlemen: 최근 급부상한 랜섬웨어 그룹으로, 공격적인 해커 모집 전략을 통해 활동량 기준 세계 2위 규모로 성장.

  • OceanLotus: 기존 외부 첩보 활동에서 국내 타겟팅으로 운영 패턴을 전환 중인 것으로 파악됨.


    • 3. 한국 관련 위협


  • 데이터 유출: Black X 그룹이 한국 및 미국 기업을 대상으로 데이터 유출을 주장하며 BreachForums에서 판매 시도.

  • 공급망 공격: 이스트시큐리티(ESRC)가 Solana 공식 SDK를 사칭한 악성 npm 패키지 9종 발견(Telegram Bot API 기반 C2 활용).

  • 금융 인프라: 디지털금융법포럼에서 에이전틱 AI 도입에 따른 금융 인프라 보안 및 인간 개입권(Human-in-the-loop)의 중요성 강조.


    • 4. 주요 CVE / 취약점


  • CVE-2026-35273: Oracle PeopleSoft, CVSS 9.8, ShinyHunters에 의해 Zero-Day로 악용됨.

  • CVE-2026-44963: Veeam Backup & Replication, RCE 취약점, 즉시 업데이트 권고.

  • CVE-2026-50751: Check Point Remote Access VPN, IKEv1 인증 우회, 현재 야생(In-the-wild)에서 활발히 악용 중.

  • CVE-2026-11645: Google Chrome V8 엔진, Out-of-bounds memory access, 긴급 업데이트 배포.

  • CVE-2026-29514: NetBox Jinja2 Sandbox Bypass, RCE로 이어질 수 있는 취약점.

    6. 즉각 대응 권고


  • 패치 적용: Veeam Backup & Replication(12.3.2.4854 이상) 및 Chrome 브라우저, Oracle PeopleSoft 등 언급된 취약점의 최신 보안 패치를 즉시 적용하십시오.

  • 공급망 점검: 개발 환경에서 사용 중인 npm 패키지 중 Solana SDK 관련 의존성을 검토하고, 비정상적인 Telegram Bot API 통신 트래픽이 있는지 모니터링하십시오.

  • VPN 보안 강화: Check Point VPN 사용 환경에서 CVE-2026-50751 악용 여부를 로그 분석을 통해 확인하고, 인증 우회 시도 차단 정책을 적용하십시오.
