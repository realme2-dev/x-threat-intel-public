1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI를 활용한 취약점 탐색 및 공격 자동화가 가속화되고 있으며, 특히 Anthropic의 'Project Glasswing'과 같은 AI 모델이 대규모 취약점 발견에 활용되는 양상을 보입니다. 공급망 공격(Supply Chain Attack)은 여전히 핵심 위협으로, npm 및 Packagist 등 오픈소스 생태계를 겨냥한 악성 패키지 유포와 GitHub 버전 태그 악용 사례가 빈번합니다. 또한, ClickFix와 같은 정교한 피싱 기법과 함께 클라우드 환경의 설정 오류 및 자격 증명 탈취를 노리는 공격이 고도화되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • Webworm (ESET): EU 정부 기관을 타겟으로 Discord 및 Microsoft Graph API를 악용하여 통신하는 고도화된 APT 캠페인 수행.

  • UNC6671 (Mandiant): 'BlackFile'이라는 명칭의 Vishing 기반 갈취 작전을 수행하며 기업 대상 금전적 이득을 노림.

  • Screening Serpens (Unit42): AppDomainManager hijacking 및 신규 RAT 변종을 사용하여 기술 및 방산 분야를 타겟팅하는 이란계 APT 그룹.

  • Kimwolf (Krebs on Security): 수백만 대의 IoT 기기를 감염시킨 대규모 봇넷 운영, 최근 관리자 체포로 활동 위축 예상.

  • System Rippers (VECERTRadar): 칠레 해군 및 군사 인프라를 타겟으로 데이터 유출을 시도하는 공격 캠페인 수행.


    • 3. 한국 관련 위협


  • ASEC 보고: 국내 화장품 및 생활화학 기업을 대상으로 한 랜섬웨어 공격(Nova) 및 금융권 대상 피싱·백도어·스틸러 공격 지속 확인.

  • 이스트시큐리티: 국세청 및 금융기관을 사칭한 PhaaS(Phishing-as-a-Service) 기반의 기업 타겟형 피싱 메일 유포 확인.


    • 4. 주요 CVE / 취약점


  • CVE-2026-26980: Ghost CMS SQL Injection, 대규모 ClickFix 캠페인에 악용 중.

  • CVE-2026-9082: Drupal, 공개 직후 공격자들에 의해 악용 시도 포착.

  • CVE-2026-46333: Linux Kernel ptrace logic flaw, 로컬 권한 상승 및 자격 증명 탈취 가능.

  • CVE-2026-3102: ExifTool, 악성 이미지를 통한 macOS 시스템 침해 가능.

    6. 즉각 대응 권고


  • 공급망 보안 강화: npm 및 Packagist 패키지 설치 시 2FA-Gated Publishing 설정 및 의존성 패키지의 무결성 검증 프로세스 도입.

  • 취약점 패치: Drupal(CVE-2026-9082) 및 Ghost CMS(CVE-2026-26980) 사용 환경은 즉시 최신 버전으로 업데이트하고, 비정상적인 JavaScript 삽입 여부 점검.

  • 클라우드/계정 보호: GitHub 등 코드 저장소에 AWS GovCloud 키와 같은 민감한 자격 증명이 노출되지 않았는지 전수 조사 및 즉각적인 키 로테이션 수행.
