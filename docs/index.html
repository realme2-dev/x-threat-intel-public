1. 현재 주요 사이버보안 트렌드


최근 사이버 위협의 핵심은 생성형 AI를 활용한 제로데이 취약점 발굴 및 공격 코드 자동화로, 공격자가 2FA 우회와 같은 고도화된 공격을 대규모로 수행하고 있습니다. 또한, 소프트웨어 공급망 공격이 지속적으로 발생하며 보안 도구(Checkmarx Jenkins AST 등)를 역이용하는 사례가 증가하고 있습니다. 더불어 Linux 환경을 타겟으로 한 LPE(Local Privilege Escalation) 취약점(Dirty Frag 등)이 활발히 악용되고 있어 엔터프라이즈 서버 보안에 비상이 걸린 상태입니다.
