1. 현재 주요 사이버보안 트렌드


최근 보안 환경은 Frontier AI 모델이 취약점 탐색 및 공격 도구로 활용되면서 공격과 방어 사이의 'Exploit Window'가 극도로 단축되고 있습니다. 특히 SaaS 및 클라우드 환경에서의 OAuth 토큰 탈취를 통한 횡적 이동(Lateral Movement)과 공급망 공격(Supply Chain Attack)이 주요 침투 경로로 자리 잡았습니다. 또한, 기존의 단순 악성코드 유포를 넘어 QEMU와 같은 가상화 도구를 악용한 방어 회피 기술이 고도화되고 있으며, OT/ICS 환경을 겨냥한 Serial-to-IP 변환기 취약점 공격이 지속적으로 보고되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • Lazarus Group (BleepingComputer): KelpDAO를 대상으로 한 $290M 규모의 대형 가상화폐 탈취 사건 배후로 지목됨.

  • Scattered Spider (SecurityWeek): 기업 내부망 침투 및 가상화폐 탈취 혐의로 주요 조직원(Tyler Buchanan)이 미국 법원에서 유죄 인정.

  • The Gentlemen (CheckPoint Research): SystemBC 프록시 봇넷을 활용하여 기업 네트워크를 공격하는 RaaS(Ransomware-as-a-Service) 그룹.

  • NoName05716 (AhnLab ASEC): 대한민국 공공기관 및 기업을 대상으로 한 DDoS 공격 활동 지속.

  • Iranian-Affiliated Actors (CISA): 미국 주요 인프라의 PLC(Programmable Logic Controllers)를 표적으로 한 공격 수행.


    • 3. 한국 관련 위협


  • 국가 배후 해킹 조직: 북한 계열 조직이 한국 정부 기관, 가상화폐 기업, 방산 및 통신 분야를 집중 타겟팅하고 있음(데일리시큐).

  • 피싱 캠페인: 급여명세서로 위장한 피싱 메일 유포 및 기업뱅킹 소프트웨어(WGear) 취약점 악용 사례 확인(이스트시큐리티).

  • 국제 훈련: 국정원이 나토 ‘Locked Shields 2026’에 6년 연속 참가하여 사이버 방어 협력 강화 중(데일리시큐).


    • 4. 주요 CVE / 취약점 (CVSS 우선)


  • CVE-2026-5760: SGLang, CVSS 9.8, Malicious GGUF 모델 파일을 통한 RCE 발생 가능.

  • CVE-2023-33538: TP-Link 라우터, Command Injection 취약점, Mirai 봇넷 악용 시도 확인.

  • CVE-2026-3888: Snap, Local Privilege Escalation(Root 권한 획득 가능).

  • CVE-2025-48757: 플랫폼 API BOLA(Broken Object Level Authorization) 취약점, 170개 이상의 프로덕션 앱 노출.

  • CVE-2026-28684: python-dotenv, Symbolic Link Following을 통한 Arbitrary File Overwrite.

    • 6. 즉각 대응 권고


  • SaaS 및 클라우드 권한 점검: Vercel 등 클라우드 서비스와 연동된 타사 애플리케이션의 OAuth 토큰 권한을 최소화하고, 비정상적인 API 접근 로그를 모니터링하십시오.

  • 기업뱅킹 소프트웨어 패치: 금융보안원 및 KISA에서 공지한 기업뱅킹 전자금융 소프트웨어(WGear) 취약점 패치를 즉시 적용하십시오.

  • AI 모델 및 모델 파일 검증: SGLang 등을 사용하는 환경에서는 신뢰할 수 없는 GGUF 모델 파일 로드를 차단하고, 샌드박스 환경에서 모델 실행을 격리하십시오.
