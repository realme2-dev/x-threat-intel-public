1. 현재 주요 사이버보안 트렌드


최근 공격자들은 공급망 공격(Supply Chain Attack)을 고도화하여 npm 패키지나 GitHub 저장소를 악용, 설치 단계에서 즉시 Infostealer를 실행하는 방식을 취하고 있습니다(TheHackerNews). 또한, AI를 활용한 피싱 메일의 comment stuffing 기법으로 탐지 시스템을 우회하거나, Wireless ADB를 악용해 모바일 기기의 권한을 탈취하는 등 공격 벡터가 다변화되고 있습니다(SANS ISC, BleepingComputer). 특히 CISA BOD 26-04와 같은 규제에 대응하기 위해 3일 이내의 신속한 취약점 패치가 강조되는 등, 공격 속도와 방어 대응 속도 간의 격차가 핵심 보안 과제로 부상했습니다(Qualys Blog).

2. 주요 위협 행위자 / 캠페인


  • Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘 등을 타겟팅(Check Point Research).

  • Armored Likho: BusySnake Stealer를 활용하여 러시아 내 조직을 타겟팅하며, AI 생성 로더를 사용하는 것이 특징(Securelist).

  • The Gentlemen: 최근 급성장 중인 랜섬웨어 그룹으로, 제휴 모델(Affiliate Model)을 통해 운영(Unit42).

  • Scattered Spider: 영국 Transport for London 공격 관련 혐의로 주요 인물들이 유죄 판결(Krebs on Security).


    • 3. 한국 관련 위협


  • 군사 인프라 타겟팅: 한국 군사 관련 사이버 공격 시도가 최근 5년 내 최고 수준에 도달함(Marcy_2022 트윗).

  • 공급망 공격: 북한 연계 해킹 그룹이 chai-as-init 등 npm 악성 패키지를 배포하여 공급망 위협을 가함(이스트시큐리티 알약).

  • 산업계 피해:住友重機械工業(스미토모 중기계 공업)의 해외 그룹사 서버에 대한 부정 접근 확인(boss_sec_labo 트윗).


    • 4. 주요 CVE / 취약점


  • CVE-2026-4480: Samba print-command injection, 인증되지 않은 RCE 유발 (cyber_advising).

  • CVE-2026-53359: Linux KVM 취약점(Januscape), 게스트 VM에서 호스트 탈출 가능 (IntCyberDigest).

  • CVE-2026-47291: Windows HTTP.sys RCE, 인증되지 않은 원격 코드 실행 (Dinosn).

  • CVE-2024-21338: Windows AppLocker 드라이버(appid.sys) 악용, 권한 상승 (cr3ghost).

  • CVE-2024-3094: XZ Utils 백도어, 공급망 보안의 취약성을 드러낸 사례 (ro0TCr4k).
