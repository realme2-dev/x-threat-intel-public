1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 모델 및 에이전트 프레임워크의 취약점을 악용하여 RCE를 수행하는 새로운 공격 벡터를 적극적으로 활용하고 있습니다. 특히 공급망 공격을 통해 신뢰받는 보안 도구 및 오픈소스 패키지(PyPI, Docker 이미지 등)를 변조하여 악성코드를 배포하는 방식이 지속적으로 관찰됩니다. 또한, Linux 커널의 LPE 취약점(Dirty Frag, Copy Fail 등)이 연이어 발견되며 인프라 단의 보안 위협이 고조되고 있으며, ShinyHunters와 같은 위협 행위자들이 교육 및 공공 부문을 타겟으로 대규모 데이터 탈취와 협박을 병행하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • ShinyHunters: Instructure(Canvas) 및 Zara를 대상으로 대규모 데이터 유출 및 랜섬웨어 협박 수행.

  • ScarCruft: Yanbian 지역을 타겟으로 게임 플랫폼을 악용한 공급망 공격 및 백도어 유포.

  • OceanLotus: PyPI 패키지를 변조하여 ZiChatBot 악성코드를 배포하는 캠페인 수행.

  • LockBit 5.0: VP Brands International을 대상으로 데이터 탈취 및 유출 협박 시도.


    • 3. 한국 관련 위협


  • 이스트시큐리티(ESRC): 2026년 1분기 랜섬웨어 동향 보고서를 통해 국내 기업 대상 랜섬웨어 피해 증가 경고 및 Wildcard DNS 기법을 악용한 급여명세서 위장 피싱 메일 주의보 발령.

  • 기타: 수집된 트윗 샘플 중 한국 관련 직접적인 공격 사례는 없으나, 글로벌 교육 플랫폼(Canvas) 해킹으로 인한 국내 대학(NUS, SIM 등)의 비밀번호 재설정 권고 조치가 확인됨.


    • 4. 주요 CVE / 취약점


    1. CVE-2026-24118: vm2 (Node.js sandbox), CVSS 9.8 (Critical), 악용 가능성 높음.
    2. CVE-2026-6973: Ivanti EPMM, 원격 코드 실행(RCE), 실제 공격 악용 확인(CISA KEV 등재).
    3. CVE-2026-0300: PAN-OS Captive Portal, 버퍼 오버플로우, 제로데이 악용 확인.
    4. CVE-2026-31431: Linux Kernel (Copy Fail), LPE, 심각한 권한 상승 취약점.
    5. CVE-2026-43284: Linux Kernel (Dirty Frag), LPE, 로컬 권한 상승 체인.
