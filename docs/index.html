1. 현재 주요 사이버보안 트렌드


최근 사이버 공격은 생성형 AI를 활용하여 취약점 탐색 및 무기화 속도를 비약적으로 단축하고 있으며, 특히 공급망 공격과 API 취약점을 악용한 자동화된 공격이 급증하고 있습니다. 또한, EDR 도입만으로는 부족한 상황에서 AI 기반의 위협 탐지 및 대응(AI-Native Security)이 필수적인 방어 전략으로 부상했습니다. 공격자들은 신뢰받는 서비스(Meta AI, npm 패키지 등)를 악용하거나, 사용자 조작을 유도하는 ClickFix/FakeUpdate 기법을 통해 보안 솔루션을 우회하는 정교한 캠페인을 전개하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • SideCopy: 파키스탄 연계 그룹으로, 아프가니스탄 재무부를 대상으로 Xeno RAT을 이용한 스피어 피싱 캠페인 수행.

  • DriveSurge: ClickFix 및 FakeUpdate 기법을 사용하여 수천 개의 사이트를 하이재킹하고 대규모 악성코드 유포.

  • UNC6671: BlackFile 랜섬웨어 및 비싱(Vishing)을 결합한 갈취 작전 수행.

  • Unknown (Red Hat NPM Supply Chain Attack): '@redhat-cloud-services' 네임스페이스를 타겟으로 Shai-Hulud 변종 웜을 배포하여 개발자 자격 증명 탈취.


    • 3. 한국 관련 위협


  • 레드햇 공급망 공격: 국내 개발 환경 및 클라우드 인프라에서도 npm 패키지 의존성 관리에 주의가 필요함.

  • 랜섬웨어 위협: ASEC 분석 결과, Midnight 랜섬웨어(Babuk 기반)가 국내 ESXi 및 NAS 환경을 타겟으로 Double Extortion 공격을 지속 중.

  • 기타: 과기정통부의 AI 3대 강국 전략 발표와 연계하여, 고성능 AI 인프라를 겨냥한 보안 주권 확보가 국가적 과제로 제시됨.


    • 4. 주요 CVE / 취약점


  • CVE-2026-49201: Acer Wave 7 Router, CVSS 10.0, 시스템 백업 타겟 및 지속적 접근 가능.

  • CVE-2026-41089: Windows Netlogon, CVSS 9.8, 도메인 컨트롤러 대상 RCE, Wild에서 악용 중.

  • CVE-2026-8732: WP Maps Pro, CVSS 9.8, 인증 없이 관리자 계정 생성, Wild에서 악용 중.

  • CVE-2026-0257: PAN-OS GlobalProtect, CVSS 미상(고위험), VPN 쿠키 위조를 통한 인증 우회, Wild에서 악용 중.

  • CVE-2026-46333: Linux Kernel ptrace, CVSS 미상(고위험), 로컬 권한 상승 및 자격 증명 노출.

    6. 즉각 대응 권고


  • Netlogon 패치: CVE-2026-41089에 대해 도메인 컨트롤러(DC) 대상 긴급 패치를 즉시 적용하고, 비정상적인 네트워크 트래픽 모니터링 강화.

  • 공급망 보안 점검: 개발 환경 내 npm 패키지 의존성을 전수 조사하고, 특히 '@redhat-cloud-services' 관련 패키지의 무결성 확인 및 자격 증명(GitHub 토큰, SSH 키) 재발급.

  • 사용자 교육: ClickFix/FakeUpdate 캠페인 대응을 위해 브라우저 내 의심스러운 팝업 및 가짜 바이러스 경고창에 대한 임직원 보안 인식 교육 실시.
