1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 에이전트(OpenClaw, LangGraph 등)의 취약점을 악용하여 코드 실행 및 데이터 탈취를 시도하는 새로운 공격 벡터를 적극 활용하고 있습니다. 특히 Oracle PeopleSoft와 같은 기업용 소프트웨어의 Zero-Day 취약점이 대규모 데이터 유출의 주요 경로로 이용되고 있으며, 공격자들은 공개된 취약점을 24시간 이내에 즉각적으로 악용하는 속도전을 펼치고 있습니다. 또한, CISA의 BOD 26-04 지침에 따라 취약점 관리의 우선순위가 더욱 엄격해졌으며, 피싱 공격은 양보다 질을 중시하는 AI 기반의 고도화된 형태로 진화하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • ShinyHunters (UNC6240): Oracle PeopleSoft Zero-Day(CVE-2026-35273)를 악용하여 대학 및 교육 기관을 타겟으로 한 데이터 탈취 및 금전 요구 캠페인 수행.

  • Void Blizzard: 러시아 국적의 Denis Obrezko와 연계된 캠페인으로, 최소 11개 미국 기업을 대상으로 한 사이버 첩보 활동 수행.

  • The Gentlemen: 최근 급부상한 랜섬웨어 그룹으로, 공격적인 해커 영입 전략을 통해 활동량을 급격히 늘리고 있음.

  • Kimsuky: 개인정보 유출 확인을 위장한 스피어피싱 메일을 통해 국내 타겟을 공격하는 APT 활동 포착.


    • 3. 한국 관련 위협


  • 큐슈전력(Kyushu Electric Power): 1,090만 명의 고객 데이터가 포함된 드라이브 분실 사고 발생(일본 기업이나 국내 보안 업계 주요 이슈로 언급).

  • 국내 온라인 교육 플랫폼: 다크웹 BreachForums에서 데이터 판매 정황 포착(AhnLab ASEC 보고).

  • Kimsuky: 국내 타겟을 대상으로 한 스피어피싱 공격 지속 탐지(이스트시큐리티 알약).


    • 4. 주요 CVE / 취약점


  • CVE-2026-35273: Oracle PeopleSoft, CVSS 9.8, ShinyHunters에 의해 실제 악용 중.

  • CVE-2026-10520: Ivanti Sentry, CVSS 10.0, Pre-auth Command Injection으로 활발히 악용 중.

  • CVE-2026-11645: Google Chrome V8 엔진, 제로데이 취약점, 긴급 업데이트 권고.

  • CVE-2026-0257: PAN-OS, 활발한 악용 사례 보고.

    6. 즉각 대응 권고


  • Oracle PeopleSoft 패치 적용: CVE-2026-35273에 대한 최신 보안 패치를 즉시 적용하고, 외부 노출 여부를 점검하십시오.

  • Ivanti Sentry 보안 점검: CISA BOD 26-04 지침에 따라 CVE-2026-10520에 대해 3일 이내 패치를 완료하고, 비정상적인 OS 명령 실행 로그를 모니터링하십시오.

  • AI 에이전트 보안 정책 강화: 사내 도입된 AI 에이전트의 접근 권한을 최소화하고, LangGraph 등 프레임워크의 Checkpointer 설정이 안전한지 검토하십시오.
