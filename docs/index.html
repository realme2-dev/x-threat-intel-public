1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI 에이전트와 자동화된 도구를 악용한 공격이 급증하고 있으며, 특히 개발 환경의 CI/CD 파이프라인과 SaaS 설정을 노리는 공격이 주를 이룹니다. 공격자들은 취약점 탐색 및 악용 단계에서 AI 모델을 적극 활용하여 공격 속도를 높이고 있으며, 공급망 공격(Supply Chain Attack)을 통해 신뢰받는 보안 도구 및 오픈소스 패키지를 변조하는 사례가 빈번합니다. 또한, 지정학적 긴장과 맞물려 국가 배후의 APT 그룹들이 critical infrastructure를 대상으로 한 파괴적 공격과 데이터 탈취를 지속하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • Silver Fox: 러시아 및 인도 기업을 대상으로 세무 당국을 사칭하여 ValleyRAT 및 ABCDoor 백도어를 유포. (Securelist)

  • UNC6692: 사회 공학적 기법을 활용하여 맞춤형 악성코드 제품군을 배포하는 다단계 침입 캠페인 수행. (Mandiant)

  • Scattered Spider: 최근 주요 멤버(Tylerb)의 유죄 판결에도 불구하고, 여전히 SaaS 환경 및 SSO 오용을 통한 고강도 갈취 공격을 지속. (SecurityWeek, Krebs on Security)

  • GopherWhisper: 몽골 정부 기관을 타겟으로 하는 중국 배후의 신규 APT 그룹. (WeLiveSecurity)


    • 3. 한국 관련 위협


  • KISA·현대차·기아 협력: 자동차 공급망 보안 강화를 위해 협력사 대상 사이버 위협 대응 역량 강화 업무협약 체결. (데일리시큐)

  • Canonical DDoS: 4월 30일 Ubuntu 운영/개발사인 Canonical을 대상으로 한 조직적인 DDoS 공격 발생으로 패치 인프라 마비. ([KR] 트윗)

  • 기업뱅킹 취약점: 기업뱅킹 전자금융 소프트웨어(WGear)의 보안 취약점 관련 대응 공지. (이스트시큐리티 알약)


    • 4. 주요 CVE / 취약점


  • CVE-2026-42779: Apache MINA AbstractIoBuffer.resolveClass() deserialization filter bypass (CVSS 9.8, RCE 가능).

  • CVE-2026-32202: Windows 원격 코드 실행 취약점, 제로데이 악용 확인 (CISA 긴급 패치 지시).

  • CVE-2026-31431: Linux kernel 취약점, 'Copy Fail'로 알려짐, 활발한 악용 확인.

  • CVE-2026-21510: Windows 관련 취약점, CVE-2026-32202와 함께 악용 보고됨.

  • CVE-2026-41940: cPanel/WHM admin interface 취약점, 인증 없이 원격 제어 가능 (Exploit 공개됨).

    6. 즉각 대응 권고


  • CVE-2026-32202 및 CVE-2026-31431 패치: CISA 지침에 따라 2주 이내에 해당 취약점에 대한 긴급 보안 업데이트를 완료하고 시스템을 재부팅하십시오.

  • cPanel/WHM 보안 점검: CVE-2026-41940 악용 시도가 활발하므로, cPanel 관리자 인터페이스에 대한 외부 접근을 차단하고 최신 보안 패치를 즉시 적용하십시오.

  • AI 브라우저 확장 프로그램 검토: 업무용 브라우저에 설치된 AI 기반 생산성 확장 프로그램이 데이터를 유출할 위험이 있으므로, 불필요한 확장 프로그램을 제거하고 권한을 재검토하십시오.
