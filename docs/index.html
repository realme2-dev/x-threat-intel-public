🟡 위협 점수: 5/10 (보통)

1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 에이전트 프레임워크(예: LangGraph)의 취약점을 악용하여 Remote Code Execution(RCE)을 수행하는 등 AI 생태계를 새로운 공격 표면으로 활용하고 있습니다. 또한, CISA의 BOD 26-04 지침에 따라 Ivanti Sentry와 같은 핵심 인프라 취약점에 대한 공격 주기가 24시간 이내로 단축되는 등 제로데이 및 N-day 취약점의 무기화 속도가 매우 빨라졌습니다. 더불어, ShinyHunters와 같은 위협 행위자들은 Oracle PeopleSoft와 같은 기업용 애플리케이션을 타겟팅하여 대규모 데이터 탈취 및 금전적 갈등을 유도하는 캠페인을 지속하고 있습니다.

2. 주요 위협 행위자 / 캠페인


- Handala: 이란 연계 해킹 그룹으로, 최근 미국 Cal Water 시설 및 FBI 드론 시스템을 해킹했다고 주장하며 지정학적 보복 성격의 공격을 수행 중입니다.
- ShinyHunters (UNC6240): Oracle PeopleSoft 취약점을 악용하여 대학 및 교육 부문을 타겟으로 데이터 유출 및 협박 캠페인을 전개하고 있습니다.
- OceanLotus: 베트남 연계 APT 그룹으로, 기존의 외부 첩보 활동에서 국내 타겟팅으로 운영 패턴을 전환하고 있습니다.
- Void Blizzard: 러시아 국적의 Denis Obrezko가 연루된 캠페인으로, 미국 기업들을 대상으로 한 광범위한 첩보 작전을 수행했습니다.

3. 한국 관련 위협


- AhnLab ASEC: 한국 온라인 교육 플랫폼의 데이터가 BreachForums에서 판매되고 있으며, Black X 그룹이 한국 기업을 대상으로 데이터 유출을 주장하고 있습니다.
- 이스트시큐리티(ESRC): Solana 공식 SDK를 사칭한 악성 npm 패키지 9종이 발견되었으며, Telegram Bot API를 C2로 활용하는 RAT이 유포되고 있습니다.
- 카스퍼스키: 한국 시장 내 OT 보안 및 AI 보안 강화를 발표하며, 2026 FIFA 월드컵 대비 멕시코 지역 등에서 Wardriving 평가를 수행하는 등 글로벌 인프라 보안 이슈가 부각되었습니다.

4. 주요 CVE / 취약점


- CVE-2026-45748: Termix SSH tunnel command injection, CVSS 9.8 (Critical), 악용 가능성 높음.
- CVE-2026-35273: Oracle PeopleSoft RCE, CVSS 9.8 (Critical), ShinyHunters에 의해 활발히 악용 중.
- CVE-2026-11849: IEI Integration Corp iRM-IEI Hardcoded Credentials, CVSS 9.8 (Critical).
- CVE-2026-10520: Ivanti Sentry OS command injection, CVSS Critical, CISA KEV 등록 및 즉각적인 패치 권고.
- CVE-2026-11645: Google Chrome V8 Out-of-bounds memory access, Critical, 제로데이 악용 사례 보고.

6. 즉각 대응 권고


- Ivanti Sentry 및 Oracle PeopleSoft 패치: CISA 지침에 따라 해당 솔루션의 최신 보안 업데이트를 즉시 적용하고, 비정상적인 OS 커맨드 실행 흔적을 모니터링하십시오.
- AI 에이전트 및 npm 패키지 검증: 사내 개발 환경에서 사용하는 AI 에이전트 프레임워크(LangGraph 등)의 의존성을 점검하고, 외부 npm 패키지 도입 시 공급망 보안 검증 절차를 강화하십시오.
- 공격 표면 관리(ASM): 외부로 노출된 관리 인터페이스(iRM-IEI 등)에 대한 접근 제어를 강화하고, Hardcoded Credentials 여부를 전수 조사하십시오.
