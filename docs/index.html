🟡 위협 점수: 5/10 (보통)

1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 WordPress Core의 RCE 취약점인 'wp2shell'과 같이 인증되지 않은 공격자가 원격 코드를 실행할 수 있는 고위험 취약점의 공개 익스플로잇이 급증하고 있습니다(BleepingComputer). 또한, AI 에이전트 및 자동화 도구를 악용한 공급망 공격과 소프트웨어 패키지(npm)를 통한 RAT 유포가 지속적으로 관찰되고 있으며(TheHackerNews), 기업의 IT 지원 시스템이나 제3자 관리 플랫폼을 경유한 데이터 유출 사고가 빈번하게 발생하고 있습니다(데일리시큐). 특히 CISA는 실제 공격에 악용된 취약점들을 긴급 보안 목록에 추가하며 신속한 패치를 강력히 권고하고 있습니다(데일리시큐).

2. 주요 위협 행위자 / 캠페인


- Cavern Manticore: 이란 연계 APT 그룹으로, 이스라엘을 타겟으로 하는 모듈형 C2 프레임워크를 운용 중임 (Check Point Research)
- Scattered Spider: 영국에서 주요 멤버들이 체포 및 실형을 선고받았으나, 여전히 기업 대상 공격의 핵심 위협으로 분류됨 (CyberScoop)
- Inc Ransomware: SonicWall SMA 장비의 Zero-Day 취약점을 연쇄적으로 악용하여 root 권한을 탈취하는 공격 수행 (DarkReading)
- GoSerpent: 동남아시아 정부 기관을 타겟으로 Stowaway RAT 등을 사용하여 데이터를 탈취하는 지속적 위협 행위자 (Securelist)

3. 한국 관련 위협


- 니치레이로지그룹(Nichirei Logistics Group) 해킹: 일본 최대 저온물류기업이 사이버 공격을 받아 KFC 등 식품 공급망에 차질 발생 (데일리시큐)
- 북한발 npm 악성 패키지: 'chai-as-init' 패키지를 통해 공급망 위협을 가한 사례 (이스트시큐리티 알약)
- 기타: 한국어 트윗에서는 주로 글로벌 취약점(wp2shell 등)에 대한 대응 가이드와 SOC 분석가 로드맵 공유가 주를 이룸.

4. 주요 CVE / 취약점


- CVE-2026-63030 / CVE-2026-60137: WordPress Core RCE 취약점, 인증 불필요, 공개 PoC 존재 (BleepingComputer)
- CVE-2026-25089 / CVE-2026-39808: Fortinet FortiSandbox OS 명령어 주입, 실제 공격 악용 확인 (데일리시큐)
- CVE-2026-58644: Microsoft SharePoint RCE, 실제 공격 악용 확인 (데일리시큐)
- CVE-2026-14266: 7-Zip RCE 취약점, 악성 압축 파일 통해 실행 가능 (BleepingComputer)

5. IOC 요약


- CVE: CVE-2026-63030, CVE-2026-60137 (BleepingComputer), CVE-2026-25089, CVE-2026-39808, CVE-2026-58644 (데일리시큐)
- 해시/도메인: 이번 수집 데이터 내 특정 파일 해시나 C2 도메인 값은 직접 명시되지 않음(공격 도구 및 패키지명 위주).

6. 즉각 대응 권고


- WordPress 업데이트: 운영 중인 모든 WordPress 사이트를 즉시 6.9.5 또는 7.0.2 버전 이상으로 업데이트하여 'wp2shell' 취약점 대응.
- Fortinet/SharePoint 패치: CISA에서 경고한 FortiSandbox 및 SharePoint 취약점(CVE-2026-25089 등)에 대해 즉각적인 보안 패치 적용.
- 공급망 검토: npm 등 외부 라이브러리 사용 시 패키지 무결성을 검증하고, 제3자 IT 지원 시스템에 대한 접근 권한 및 로그 모니터링 강화.
