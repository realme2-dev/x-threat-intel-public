1. 현재 주요 사이버보안 트렌드


최근 공격 생태계는 LLM(Large Language Model) 에이전트를 활용한 자동화된 공격과 AI 기반 해킹 도구의 확산이 핵심 트렌드로 자리 잡았습니다. 특히, 공격자들이 AI 모델의 취약점을 이용해 코드 실행을 유도하거나, AI 에이전트를 통해 포스트 익스플로잇(Post-Exploitation) 단계를 자동화하는 사례가 급증하고 있습니다. 또한, PAN-OS와 같은 엔터프라이즈 보안 장비의 인증 우회 취약점이 실시간으로 악용되고 있으며, 공급망 공격과 결합된 복합형 악성코드(Stealer, RAT, Ransomware 통합)의 위협이 지속되고 있습니다.

2. 주요 위협 행위자 / 캠페인


- Kimsuky (데일리시큐, AhnLab ASEC): 한국 군·기업 관계자를 대상으로 보안 프로그램 및 화상회의 서비스 사칭 공격 수행.
- UNC6671 (Mandiant): 'BlackFile'이라는 명칭의 비싱(Vishing) 및 갈취 캠페인 운영.
- Screening Serpens (Unit42): AppDomainManager Hijacking 및 신규 RAT 변종을 사용하여 기술·방산 분야 타겟팅.
- ShinyHunters (Malwarebytes): Carnival 데이터 유출 사건의 배후로 지목됨.
- Handala (트위터/X): 홀로코스트 피해자 지원 센터 대상 데이터 유출 및 사이버 공격 수행.

3. 한국 관련 위협


- Kimsuky 활동: 3~4월 한국 군·기업 관계자를 겨냥한 정교한 피싱 공격(JSONPing, Webex 사칭) 포착.
- 카드사 사칭: 국내 유명 카드사 보안 메일로 위장한 악성 LNK 파일 유포 정황 확인.
- 기타: 일본 정부 기관 및 교육 기업 데이터 유출 사건이 국내 보안 커뮤니티에서 주요 이슈로 다뤄짐.

4. 주요 CVE / 취약점


- CVE-2026-31525: Linux BPF subsystem, CVSS 7.8, 권한 상승 및 컨테이너 탈출 가능.
- CVE-2026-0257: Palo Alto Networks PAN-OS GlobalProtect, CVSS 7.8, Active Exploitation(실제 악용 중).
- CVE-2026-39987: Marimo, LLM 에이전트 기반 포스트 익스플로잇에 활용됨.
- CVE-2026-46333: Linux kernel ptrace path, 로컬 권한 상승 및 자격 증명 탈취 가능.
- CVE-2026-45585: Windows Disk Encryption Bypass (YellowKey), CVSS 6.8.
