1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 에이전트의 취약점을 노린 'AI Supply Chain' 공격과 오픈소스 생태계의 CI/CD 파이프라인을 악용하는 공급망 공격에 집중하고 있습니다. 특히 Cisco SD-WAN과 같은 네트워크 인프라 장비의 Zero-day 취약점을 활용해 관리자 권한을 탈취하는 사례가 빈번하며, 악성코드 분석을 방해하기 위해 AI 분석 도구를 교란하는 'Forbidden Text' 삽입 기법이 등장했습니다. 또한, Amadey 및 StealC와 같은 대규모 인포스틸러 네트워크가 사법 당국에 의해 타격받는 등 인프라 중심의 대응과 공격이 공존하는 양상을 보입니다.

2. 주요 위협 행위자 / 캠페인


  • The Gentlemen: EDR killer framework를 유지하며 RaaS(Ransomware-as-a-Service) 활동을 전개, 최근 가장 활발한 랜섬웨어 그룹 중 하나로 부상.

  • Scattered Spider: Transport for London 공격 등 대규모 인프라 타격, 최근 재판 과정에서 범죄 사실 인정.

  • FishMonger: SprySOCKS for Windows 백도어를 활용하여 커널 드라이버 수준의 은밀한 공격 수행.

  • Kimsuky: 한국 관련 기관 및 보안 담당자를 대상으로 스피어 피싱 공격 지속.

  • Popa Botnet: 수백만 대의 Android 기반 TV 박스를 감염시켜 광고 사기 및 데이터 탈취에 활용.


    • 3. 한국 관련 위협


  • Kimsuky: 보안 담당자를 대상으로 개인정보 유출을 사칭한 스피어 피싱 공격 수행.

  • 공급망 공격: 북한 연계 해커가 유명 테스트 프레임워크인 Chai.js의 플러그인으로 위장한 악성 npm 패키지(chai-as-init) 배포.

  • 산업 협력: 한국정보산업연합회와 일본 JISA 간 AI 비즈니스 협력 및 스피어에이엑스(SpearAX)와 미국 블레이즈(Blaize)의 엣지 AI 컴퓨팅 협력 등 AI 보안 관련 산업 동향 확인.


    • 4. 주요 CVE / 취약점


  • CVE-2026-48908: Joomla SP Page Builder, CVSS 10.0, RCE 가능, 현재 악용 중.

  • CVE-2026-20245: Cisco Catalyst SD-WAN, 심각도 높음, Zero-day 악용으로 root 권한 탈취 확인.

  • CVE-2026-41089: Windows Netlogon, CLDAP Stack Buffer Overflow, RCE 가능.

  • CVE-2026-45504: Microsoft Exchange, File Read 취약점.

  • CVE-2024-40766: 설정 오류로 인한 보안 취약점, 패치 이후에도 설정 미흡으로 악용 가능.

    6. 즉각 대응 권고


  • 네트워크 장비 점검: Cisco SD-WAN 및 Lantronix EDS5000 시리즈 장비의 펌웨어 업데이트 및 관리자 계정 접근 로그를 즉시 감사하십시오.

  • 개발 환경 보안: CI/CD 파이프라인 내 오픈소스 패키지(npm 등) 도입 시 출처 검증을 강화하고, chai-as-init 등 의심스러운 패키지 사용 여부를 전수 조사하십시오.

  • 피싱 대응: 보안 담당자를 대상으로 한 '개인정보 유출' 사칭 피싱 메일이 유포 중이므로, 첨부파일 실행 전 샌드박스 분석 및 발신자 확인 절차를 준수하십시오.
