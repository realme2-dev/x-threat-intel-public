1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 코딩 도구 및 에이전트의 취약점을 악용하여 공급망 공격을 수행하는 등 공격 벡터를 지능화하고 있습니다. 특히 러시아 정보기관 연계 그룹은 Signal 백업 키 탈취와 같은 사용자 인증 정보 중심의 타겟팅을 강화하고 있으며, 클라우드 자격 증명 탈취를 위한 악성 저장소 배포가 활발합니다. 또한, 취약점 공개와 동시에 PoC가 GitHub 등에 즉각 배포되는 'Zero-day' 및 'CVE' 악용 속도가 빨라져 방어 측의 신속한 패치 대응이 더욱 중요해졌습니다.

2. 주요 위협 행위자 / 캠페인


- Russian Intelligence Services (SSU/FBI/CISA): Signal 백업 복구 키 탈취 및 피싱을 통한 메시징 자격 증명 탈취 캠페인 수행.
- Gamaredon (ESET Research): 터널링, Dead drops, 신규 동맹을 활용한 C&C 인프라 은닉 및 데이터 유출.
- CL-STA-1062 (Unit 42): 동남아시아 정부 및 핵심 인프라를 타겟으로 TinyRCT 백도어 활용.
- Kimsuky (이스트시큐리티 ESRC): 개인정보 유출을 사칭한 스피어 피싱으로 보안 담당자 타겟팅.

3. 한국 관련 위협


- Kimsuky: 보안 담당자를 대상으로 개인정보 유출을 사칭한 스피어 피싱 공격 지속 확인.
- 공급망 공격: Chai.js 플러그인으로 위장한 악성 npm 패키지(chai-as-init) 유포 사례 확인.
- 기타: 인도 언론 등에서 파키스탄 해커에 의한 인도 웹사이트 해킹 사례가 언급되었으나, 한국 내 직접적인 대규모 침해 사고는 이번 수집 데이터상 보고되지 않음.

4. 주요 CVE / 취약점


- CVE-2026-23918: Apache httpd 2.4.66, CVSS 8.8 (High), DoS 및 RCE 가능성.
- CVE-2026-43503: Linux Kernel (DirtyClone), 권한 상승 가능, 로컬 익스플로잇 공개.
- CVE-2026-48908: Joomla SPB, RCE, 실환경 악용 중.

- CVE-2026-58051: libssh2, Uninitialized Pointer Free, High.
