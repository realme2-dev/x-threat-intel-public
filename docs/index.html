1. 현재 주요 사이버보안 트렌드


최근 위협 환경은 AI 에이전트의 보안 역량을 강화하려는 방어 측의 움직임과, 이를 우회하거나 악용하려는 공격자 간의 기술 경쟁이 심화되고 있습니다. 특히 WordPress 플러그인 및 오픈소스 패키지(npm)를 겨냥한 공급망 공격이 활발하며, FortiGate와 같은 네트워크 장비의 취약점을 이용한 대규모 계정 탈취 캠페인이 지속적으로 보고되고 있습니다. 또한, EDR을 무력화하는 전용 프레임워크가 RaaS 조직에 의해 배포되는 등 공격의 정교함과 자동화 수준이 급격히 높아지고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • The Gentlemen (Krebs on Security, WeLiveSecurity): GentleKiller EDR 프레임워크를 활용하여 보안 프로세스를 무력화하는 RaaS 조직으로, 최근 가장 활발하게 활동 중입니다.

  • Sapphire Sleet (BlueNoroff) (BleepingComputer, 데일리시큐): Mastra AI 생태계를 겨냥한 공급망 공격을 수행하며, npm 패키지에 악성코드를 삽입하여 개발자 인증 정보 및 가상화폐를 탈취합니다.

  • Evil Corp (CyberScoop, Malwarebytes): SocGholish 봇넷을 운영하며 브라우저 업데이트를 위장한 악성코드 유포를 주도했으나, 최근 대규모 단속으로 15,000개 이상의 사이트가 정화되었습니다.

  • TeamPCP (CyberScoop): 오픈소스 소프트웨어 생태계의 신뢰 모델을 악용하여 개발 속도를 우선시하는 환경을 타겟팅하는 공격 그룹입니다.


    • 3. 한국 관련 위협


  • 북한 연계 해킹 조직: 이스트시큐리티(ESRC)는 북한 해킹 조직이 npm 패키지(chai-as-init)를 위장하여 공급망 공격을 시도하고, 김수키(Kimsuky)가 보안 담당자를 대상으로 스피어 피싱을 수행하는 정황을 확인했습니다.

  • 금융권 보안 이슈: 안랩(AhnLab ASEC) 보고서에 따르면, 2026년 5월 국내 금융권을 대상으로 한 피싱 공격이 급증하고 있으며, 인포스틸러 및 랜섬웨어 유포가 지속되고 있습니다.


    • 4. 주요 CVE / 취약점


  • CVE-2026-4020: Gravity SMTP WordPress Plugin, API Key 및 OAuth 토큰 유출, 현재 활발히 악용 중.

  • CVE-2026-20230: Cisco Unified Communications Manager, SSRF 및 파일 쓰기 취약점을 통한 RCE 체인 공격 가능.

  • CVE-2026-7524/7700/7687: Langflow, 다중 벡터 RCE 공격 프레임워크로 악용 가능.

  • CVE-2026-46461: Dell Server Hardware Manager, 권한 상승 취약점 (CVSS 7.8).

  • CVE-2026-35273: Oracle PeopleSoft Enterprise PeopleTools, 인증되지 않은 공격자의 접근 허용.

    6. 즉각 대응 권고


  • Gravity SMTP 업데이트: 사용 중인 WordPress 환경에서 Gravity SMTP 플러그인을 최신 버전으로 즉시 업데이트하고, 노출된 API Key 및 OAuth 토큰을 즉시 교체(Rotate)하십시오.

  • FortiGate 보안 점검: FortiBleed 캠페인 대응을 위해 모든 FortiGate 장비의 관리자 계정 비밀번호를 변경하고, VPN 접근 제어 정책을 강화하십시오.

  • npm 공급망 모니터링: 개발 환경에서 사용 중인 오픈소스 패키지(특히 AI/ML 관련)의 무결성을 검증하고, 출처가 불분명한 패키지 설치를 제한하십시오.
