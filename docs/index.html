1. 현재 주요 사이버보안 트렌드


최근 사이버 위협은 생성형 AI를 활용한 공격의 정교화와 자율화가 핵심이며, 특히 AI 에이전트가 공격 체인 전반(정찰, 침투, 데이터 탈취)을 자동화하는 양상을 보입니다. 공급망 공격(Supply Chain Attack)은 NPM 패키지 및 GitHub Action을 타겟으로 하여 개발자 환경을 직접적으로 위협하고 있으며, 기업의 클라우드 인프라와 AI 모델의 취약점을 노리는 공격이 급증하고 있습니다. 또한, 랜섬웨어 그룹은 단순 암호화를 넘어 데이터 유출 협박(Double Extortion)과 결합하여 공격의 파괴력을 극대화하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • TA4922 (DarkReading): 동아시아를 넘어 유럽 및 아프리카로 피싱 캠페인을 확대 중인 다목적 사이버 범죄 그룹.

  • UNC6671 (Mandiant): 'BlackFile'이라는 명칭으로 비싱(Vishing) 및 강탈 작전을 수행하는 위협 행위자.

  • IronWorm (DarkReading/Unit42): NPM 공급망을 타겟으로 Rust 기반 악성코드를 유포하여 자격 증명을 탈취하고 확산하는 캠페인.

  • Handala (Recorded Future): 이란 MOIS와 연계된 그룹으로, 사이버 공격과 물리적 사보타주를 결합한 하이브리드 작전 수행.


    • 3. 한국 관련 위협


  • 비ジュアルアーツ(Visual Arts): 내부 데이터 유출 및 마이넘버 포함 개인정보 탈취 피해 발생.

  • 자동화 설비 기업: Qilin 랜섬웨어 그룹의 타겟이 되어 데이터 강탈 피해 발생.

  • 성형외과: Black X 그룹에 의한 내부 데이터 유출 주장.

  • 금융기관: 아카마이 보고서에 따르면 아태지역 금융기관을 대상으로 한 Layer 7 DDoS 공격이 집중됨.

  • 무인양품 등 17개사: polyfill[.]io 관련 공급망 공격으로 인한 불신 인증 화면 노출 피해.


    • 4. 주요 CVE / 취약점


  • CVE-2026-20230: Cisco Unified Communications Manager, 미인증 원격 코드 실행(RCE) 및 Root 권한 획득 가능, PoC 공개로 악용 위험 매우 높음.

  • CVE-2026-3300: Everest Forms Pro, 미인증 RCE, 대규모 악용 사례 확인됨.

  • CVE-2026-21404: NAVTOR NavBox, SOAP 메서드 무단 접근 및 운영 중단 위험.

  • CVE-2013-3900: MS Word 변조 악성코드, 디지털 서명 검증 우회에 악용됨.
