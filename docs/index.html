🟠 위협 점수: 8/10 (높음)

1. 현재 주요 사이버보안 트렌드


최근 보안 위협은 AI를 활용한 공격 자동화와 공급망 공격의 고도화가 두드러지며, 특히 LLM을 악용한 피싱 메일 작성 및 코드 취약점 탐색이 활발해지고 있습니다(DarkReading). 또한, 공격자들은 기존의 기술적 취약점 공격보다 사용자의 부주의를 유도하는 게임 파일 위장이나 피싱 메일 등 사회공학적 기법을 선호하는 경향이 뚜렷합니다(BleepingComputer). 더불어, CISA는 KEV(Known Exploited Vulnerabilities) 목록을 통해 알려진 취약점에 대한 즉각적인 패치를 강제하고 있으며, 기업들은 AI 거버넌스 부재로 인한 보안 공백을 메우기 위한 전략적 대응이 요구되는 시점입니다(Crowdstrike Blog).

2. 주요 위협 행위자 / 캠페인


  • Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘 등 주요 타겟을 공격(Check Point Research).

  • GoSerpent: 동남아시아 정부 기관을 타겟으로 하는 백도어 캠페인으로, Stowaway RAT 등을 활용해 데이터 탈취 수행(Securelist).

  • OkoBot: 암호화폐 사용자를 타겟으로 하는 악성코드 프레임웨어로, 브라우저 모니터링 및 시드 구문 탈취(Securelist).

  • Scattered Spider: 2024년 Transport for London 해킹 사건의 주범으로, 최근 5.5년형 선고(TheHackerNews).


    • 3. 한국 관련 위협


  • 이스트시큐리티 ESRC: 국세청 사칭 피싱 메일을 통한 원격제어 악성코드 유포 주의보 발령(이스트시큐리티 알약).

  • 공급망 공격: Chai.js 플러그인으로 위장한 북한발 npm 악성 패키지 'chai-as-init' 유포 확인(이스트시큐리티 알약).

  • 오픈소스 서밋 코리아: 8월 13일~14일 서울 개최 예정, AI 및 보안 기술 공유(데일리시큐).


    • 4. 주요 CVE / 취약점


  • CVE-2026-58644: Microsoft SharePoint Server RCE, 심각도 Critical (CVSS 9.8), CISA KEV 등록 및 활발히 악용 중(TheHackerNews).

  • CVE-2026-42533: NGINX RCE, 심각도 Critical (CVSS 9.2), 원격 코드 실행 가능(데일리시큐).
