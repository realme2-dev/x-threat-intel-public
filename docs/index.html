제공된 위협 인텔리전스 데이터를 바탕으로 분석한 결과입니다.

1. 현재 주요 사이버보안 트렌드


최근 공격자들은 Microsoft Teams와 같은 협업 툴을 악용한 사회공학적 기법(Social Engineering)과 npm 패키지 오염을 통한 공급망 공격(Supply Chain Attack)을 결합하여 개발자 및 기업 내부망에 침투하고 있습니다. 특히 AI를 활용한 취약점 탐색 및 공격 자동화 시도가 증가하고 있으며, 클라우드 환경의 설정 오류(Configuration Drift)와 API 취약점을 노린 공격이 주요 위협으로 부상했습니다. 또한, 모바일 기기의 알림 데이터베이스 등 포렌식 잔여 데이터를 악용하는 사례가 확인되어 엔드포인트 보안 정책의 재점검이 요구됩니다.

2. 주요 위협 행위자 / 캠페인


  • UNC6692: Microsoft Teams를 통해 IT Helpdesk를 사칭, 커스텀 악성코드(SNOW Malware)를 배포하는 사회공학적 공격 수행.

  • Scattered Spider: 최근 주요 멤버(Tylerb)가 기소되었으나, 여전히 대규모 자격 증명 탈취 및 기업 침해 사고의 핵심 위협으로 활동 중.

  • Checkmarx Supply Chain Campaign: Bitwarden CLI npm 패키지를 오염시켜 개발자 자격 증명을 탈취하는 공급망 공격 진행.

  • Chinese APT (Mongolia 타겟): Microsoft Outlook, Slack, Discord, file.io 등 클라우드 툴을 C2 채널로 활용하여 몽골 대상 정보 수집 활동.


    • 3. 한국 관련 위협


  • 기업뱅킹 소프트웨어(WGear) 취약점: 금융보안원 및 KISA를 통해 기업뱅킹 전자금융서비스의 엑셀 대용량 처리 프로그램 취약점 대응 권고.

  • 피싱 메일: Wildcard DNS 기법을 악용한 급여명세서 위장 피싱 메일이 국내에서 유포 중.

  • APT 공격: 2026년 3월 국내 타깃 APT 공격 동향 보고서에 따르면, .lnk 파일을 이용한 파워쉘(PowerShell) 실행 방식이 주된 유포 매개체로 확인됨.


    • 4. 주요 CVE / 취약점 (CVSS 우선)


  • CVE-2026-3461: WooCommerce Visa 결제 플러그인, CVSS 9.8 (Critical), 인증 우회 및 관리자 권한 탈취 가능.

  • CVE-2026-6942: radare2-mcp, CVSS 9.8 (Critical), OS Command Injection.

  • CVE-2026-40372: ASP.NET Core, CVSS 9.1 (Critical), 권한 상승 취약점.

  • CVE-2026-26210: KTran, CVSS 9.3 (Critical), 원격 코드 실행(RCE).

  • CVE-2026-28950: Apple iOS/iPadOS Notification Services, 심각도 높음, 삭제된 알림 데이터 복구 취약점.

    6. 즉각 대응 권고


  • 공급망 보안 강화: Bitwarden CLI 등 개발 환경에서 사용하는 npm 패키지 및 의존성 라이브러리의 무결성을 즉시 검증하고 최신 버전으로 업데이트하십시오.

  • 사회공학적 공격 대비: Microsoft Teams 등 협업 툴을 통한 IT 지원 사칭 메시지에 주의하고, 외부 요청 시 공식 채널을 통한 재확인 프로세스를 의무화하십시오.

  • 패치 및 설정 검토: ASP.NET Core 및 기업뱅킹 관련 보안 업데이트를 즉시 적용하고, 모바일 기기 내 민감한 알림 데이터가 잔존하지 않도록 보안 정책을 강화하십시오.
