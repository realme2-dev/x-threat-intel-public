1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI Agent를 활용한 공격 자동화가 실전 단계에 진입하며, 정찰부터 데이터 암호화까지 전 과정을 자율적으로 수행하는 공격이 관찰되고 있습니다. 특히 LLM의 환각(Hallucination) 현상을 악용한 'Phantom Squatting'과 같은 공급망 공격이 증가하고 있으며, ClickFix와 같은 고도화된 사회공학적 기법이 악성코드 배포의 표준으로 자리 잡고 있습니다. 또한, 취약점 연구자를 타겟으로 한 위장 PoC 코드 배포 등 공격자의 타겟팅이 더욱 정교해지고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • JADEPUFFER (Sysdig): Langflow RCE 취약점을 악용하여 AI Agent 기반의 데이터베이스 랜섬웨어 공격을 자율적으로 수행.

  • INC & Lynx (TheHackerNews/BleepingComputer): FortiBleed 캠페인을 통해 탈취한 Fortinet 자격 증명을 활용하여 네트워크 침투 및 랜섬웨어 공격 수행.

  • TAG-182 (Recorded Future): MarkiRAT 악성코드를 사용하여 이란 내 타겟 대상 사이버 감시 및 정보 탈취 수행.

  • Scattered Spider (BleepingComputer/Krebs): 주요 인프라 및 기업 대상 데이터 유출 및 랜섬웨어 공격, 최근 핵심 조직원 미국 인도 및 기소.

  • Kimsuky (이스트시큐리티): 보안 담당자를 대상으로 개인정보 유출을 사칭한 스피어 피싱 공격 수행.


    • 3. 한국 관련 위협


  • Settra: 한국 산업장비 제조 대기업의 해외 법인 데이터 유출 주장 (AhnLab ASEC).

  • 공급망 공격: Chai.js 플러그인으로 위장한 악성 npm 패키지 chai-as-init 배포 (이스트시큐리티).

  • APT 공격: 2026년 5월 국내 타겟 APT 공격 동향 보고서 발행, 지속적인 은밀한 표적 공격 확인 (AhnLab ASEC).


    • 4. 주요 CVE / 취약점


  • CVE-2026-45659: Microsoft SharePoint RCE, CISA KEV 등재 및 활발히 악용 중.

  • CVE-2026-43503: Ubuntu DirtyClone Kernel Flaw, 긴급 패치 권고.

  • CVE-2026-4372: Hugging Face Transformers RCE, AI 모델 로딩 시 악성 config.json으로 시스템 장악 가능.

  • CVE-2026-44937: Rancher Fleet Webhook RCE/DoS, 설정 미흡 시 악용 가능.

  • CVE-2026-8147: MLflow Trace API 권한 우회, 인증 활성화 환경에서도 취약.
