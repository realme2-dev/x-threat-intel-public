1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 Agentic AI를 활용한 공격 자동화가 실전 단계에 진입하며 공격자와 방어자 간의 격차가 급격히 벌어지고 있습니다. 특히 LLM을 이용한 취약점 탐색 및 공격 코드 생성, Phantom Squatting과 같은 공급망 공격 벡터가 고도화되고 있습니다. 또한, 기존의 소프트웨어 취약점 악용뿐만 아니라 사회공학적 기법과 결합된 PhaaS(Phishing-as-a-Service) 및 악성 패키지 배포가 지속적으로 증가하는 추세입니다.

2. 주요 위협 행위자 / 캠페인


  • North Korean Hackers (TheHackerNews): npm, Packagist, Go 등 오픈소스 생태계에 108개의 악성 패키지 및 브라우저 확장 프로그램을 배포하는 'PolinRider' 캠페인 수행.

  • JadePuffer (BleepingComputer): LLM 에이전트를 활용하여 공격 전 과정을 자동화한 최초의 랜섬웨어 사례로 보고됨.

  • ShinyHunters (SecurityWeek): Medtronic의 기업 IT 시스템에 침투하여 380만 명의 개인 및 의료 정보를 탈취.

  • TAG-182 (Recorded Future): MarkiRAT 악성코드를 활용하여 이란 내 타겟을 대상으로 사이버 감시 작전 수행.

  • Scattered Spider (Krebs on Security): Transport for London 공격 관련 주요 인물들이 유죄를 인정하거나 미국으로 인도됨.


    • 3. 한국 관련 위협


  • Settra: 한국의 산업장비 제조 대기업 해외 법인 데이터 유출 주장 (AhnLab ASEC).

  • Kimsuky: 보안 담당자를 대상으로 개인정보 유출을 사칭한 스피어 피싱 공격 지속 (이스트시큐리티 알약).

  • 공급망 공격: 'chai-as-init' 악성 npm 패키지 배포 등 국내 개발 환경을 겨냥한 공급망 위협 확인 (이스트시큐리티 알약).


    • 4. 주요 CVE / 취약점

  • CVE-2026-47896: Apache Lucene.Net, CVSS 9.x(추정), 인증되지 않은 임의 파일 읽기.

  • CVE-2026-57270: GeoVision GeoWebplayer, CVSS 8.3, OOB 액세스 취약점 (미패치).

  • CVE-2026-41096: Windows DNS Client, CVSS 8.x(추정), 원격 코드 실행(RCE) 가능.
