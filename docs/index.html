1. 현재 주요 사이버보안 트렌드


최근 공격자들은 단순히 데이터를 암호화하는 랜섬웨어에서 벗어나, 데이터 탈취 및 유출을 통한 금전적 갈취(Extortion)로 전략을 전환하고 있습니다. 특히 AI 에이전트와 LLM을 활용한 취약점 탐색 및 자동화된 공격이 증가하고 있으며, Pwn2Own Berlin 2026에서 입증되었듯 Microsoft Exchange, Windows 11 등 핵심 인프라의 Zero-day 취약점이 활발히 악용되고 있습니다. 또한, 소프트웨어 공급망 공격(node-ipc 등)과 사회공학적 기법을 결합한 정교한 침투 시도가 지속적으로 관찰됩니다.

2. 주요 위협 행위자 / 캠페인


- Turla: Kazuar backdoor를 모듈형 P2P botnet으로 진화시켜 지속적인 접근(Persistence) 확보.
- UNC6671: BlackFile vishing extortion operation을 통해 기업 대상 금전 갈취 캠페인 수행.
- ShinyHunters: Instructure Canvas 등 교육 기술 플랫폼을 타겟으로 한 데이터 유출 및 협박 활동.
- Kimsuky: PebbleDash 기반 도구와 AppleSeed malware cluster를 활용하여 조직 대상 정보 탈취.
- FrostyNeighbor: 업데이트된 compromise chain을 통해 사이버 스파이 활동 강화.

3. 한국 관련 위협


- Pwn2Own Berlin 2026: 한국 연구자가 NVIDIA 인프라 공격에 성공하여 상위권 성적 기록.
- 이스트시큐리티 ESRC: 국세청 및 금융기관을 사칭한 PhaaS(Phishing-as-a-Service) 기반의 기업 타겟형 피싱 메일 유포 확인.
- AhnLab ASEC: 국내 의료 초음파 장비 제조 기업의 데이터 유출 사례 보고.

4. 주요 CVE / 취약점


- CVE-2026-20182: Cisco Catalyst SD-WAN Controller, CVSS 10.0, 국가 배후 그룹에 의해 활발히 악용 중.
- CVE-2026-26030: Microsoft Semantic Kernel (Python SDK), CVSS 9.8, Prompt injection을 통한 RCE 가능.
- CVE-2026-42897: Microsoft Exchange Server, 심각도 높음, OWA 관련 스푸핑/XSS 취약점으로 실악용 중.
- CVE-2026-42945: NGINX rewrite module, Heap buffer overflow, RCE/DoS 가능.
- CVE-2025-68670: xrdp server, Pre-auth RCE, 보안 업데이트 권고.

6. 즉각 대응 권고


- Cisco SD-WAN 및 Exchange Server 패치: CVE-2026-20182 및 CVE-2026-42897에 대한 벤더사 보안 권고안을 즉시 적용하고, 비정상적인 관리자 로그인 로그를 모니터링하십시오.
- AI/LLM 기반 서비스 보안 점검: Semantic Kernel 등 AI 프레임워크를 사용하는 환경에서 Prompt injection을 통한 RCE 위험을 인지하고, 입력값 검증 및 필터링 정책을 강화하십시오.
- 피싱 메일 대응: 국세청 및 금융기관을 사칭한 피싱 메일이 유포 중이므로, 임직원 대상 사회공학적 기법 주의 교육 및 이메일 첨부파일 실행 통제를 강화하십시오.
