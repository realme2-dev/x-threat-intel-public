1. 현재 주요 사이버보안 트렌드


최근 사이버 위협은 AI 에이전트 생태계의 취약점을 악용하여 보안 스캔을 우회하는 공급망 공격이 급증하고 있으며, 특히 AI 모델의 가드레일을 무력화하는 Jailbreak 시도가 빈번합니다. FortiGate와 같은 네트워크 보안 장비의 자격 증명을 탈취하는 대규모 캠페인(FortiBleed)이 지속되고 있으며, CI/CD 파이프라인의 취약점을 이용한 악성 Pull Request 삽입 공격이 개발 워크플로우를 위협하고 있습니다. 또한, 연방 정부 차원의 Post-Quantum Cryptography(PQC) 전환이 가속화되는 등 차세대 암호 체계 대응이 주요 보안 의제로 부상했습니다.

2. 주요 위협 행위자 / 캠페인


  • The Gentlemen (Krebs on Security): EDR Killer 프레임워크를 활용하여 보안 솔루션을 무력화하고 랜섬웨어 공격을 수행하는 신흥 범죄 그룹.

  • Scattered Spider (Krebs on Security): Transport for London 등을 공격한 혐의로 주요 조직원이 유죄 판결을 받았으며, 사회공학적 기법을 통한 초기 침투에 능함.

  • FishMonger (WeLiveSecurity): SprySOCKS 백도어와 커널 드라이버를 활용하여 고도화된 스텔스 공격을 수행하는 행위자.

  • Kimsuky (AhnLab ASEC): 개인정보 유출을 사칭한 스피어 피싱 공격을 통해 국내 타겟을 지속적으로 공격.


    • 3. 한국 관련 위협


  • 북한 연계 공격: 이스트시큐리티 ESRC 보고에 따르면, 북한발 악성 npm 패키지(chai-as-init)가 배포되었으며, Kimsuky는 보안 담당자를 대상으로 한 스피어 피싱 공격을 지속 중임.

  • 금융권 위협: 2026년 5월 기준 국내 금융권 대상 피싱 공격이 2.3으로 급증했으며, Dropper/Downloader 및 Backdoor 유포가 활발함.


    • 4. 주요 CVE / 취약점


  • CVE-2026-44631: Apache HTTP Server, CVSS 9.8 (Critical), 메모리 오염 및 코드 실행 가능.

  • CVE-2026-20230: Cisco Unified Communications Manager, High-Severity, SSRF 취약점, 현재 공격 악용 중.

  • CVE-2026-41089: Windows Netlogon, RCE, CLDAP 스택 버퍼 오버플로우.

  • CVE-2026-47729: Squid Proxy, 히프 과읽기(Heap Over-read), 인증 정보 및 API 키 유출 가능성.

  • CVE-2026-41947: Dify AI Platform, 데이터 노출 취약점, 타 테넌트 정보 접근 가능.

    • 6. 즉각 대응 권고


  • 네트워크 장비 점검: FortiGate 및 Cisco Unified CM 서버의 패치 상태를 즉시 확인하고, 관리자 계정의 자격 증명 재설정 및 MFA 적용을 강제할 것.

  • 개발 환경 보안: npm 패키지(chai-as-init 등) 및 CI/CD 파이프라인에 유입되는 Pull Request에 대한 자동화된 코드 검증 프로세스를 강화할 것.

  • AI 에이전트 가드레일: 사내 도입된 AI 에이전트 및 플랫폼(Dify 등)의 권한 설정을 재검토하고, 민감 데이터 접근 제어 정책을 최신화할 것.
