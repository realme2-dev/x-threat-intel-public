1. 현재 주요 사이버보안 트렌드


최근 사이버 공격은 생성형 AI를 활용하여 공격 진입 장벽을 낮추고, 취약점 탐색 및 피싱 캠페인의 자동화를 가속화하는 추세입니다. 특히 AI 에이전트가 직접 취약점을 탐색하거나 피싱 통로로 악용되는 사례가 증가하고 있으며, 오픈소스 공급망 공격과 결합하여 기업의 클라우드 인프라를 타겟팅하고 있습니다. 또한, VPN 및 인증 우회 취약점(CVE-2026-0257 등)을 통한 초기 침투와 스틸러(Stealer), 랜섬웨어(Ransomware)가 결합된 복합형 악성코드 위협이 지속적으로 보고되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • UNC6671: Vishing을 결합한 BlackFile Extortion Operation 수행.

  • Screening Serpens (Iranian APT): AppDomainManager hijacking 및 신규 RAT 변종을 사용하여 기술 및 방산 분야 타겟팅.

  • Cloud Atlas: 러시아/벨라루스 공공 및 외교 기관 타겟, ReverseSocks 및 PowerCloud 도구 사용.

  • GREYVIBE: AI 기반 사이버 공격 기술을 활용하여 우크라이나 인프라 공격.


    • 3. 한국 관련 위협


  • Kimsuky: 국내 카드사 보안 메일로 위장한 악성 LNK 파일 유포 캠페인 확인 (AhnLab ASEC).

  • 기타: 국내 유명 카드사 사칭 악성 파일 유포 및 암호화 메신저 Signal 백업 복구키 탈취 피싱 공격이 국내 이용자를 대상으로 확산 중 (데일리시큐).


    • 4. 주요 CVE / 취약점


  • CVE-2026-0257: PAN-OS GlobalProtect Authentication Bypass, 심각도 높음, Active Exploitation.

  • CVE-2026-46333: Linux Kernel ptrace Path Local Root Privilege Escalation.

  • CVE-2026-40933: Flowise RCE, PoC 공개로 인한 즉각적인 위험.

  • CVE-2026-8732: WP Maps Pro, 미인증 관리자 계정 생성 취약점 악용 중.

  • CVE-2026-35616: FortiClient EMS, EKZ Infostealer 배포에 악용.
