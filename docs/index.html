제공해주신 위협 인텔리전스 데이터를 분석한 결과입니다.

1. 현재 주요 사이버보안 트렌드


최근 보안 업계는 AI 모델(Anthropic의 Mythos 등)을 활용한 대규모 취약점 탐색과 이를 악용한 공격이 급증하는 추세입니다. 특히 npm, PyPI, Crates.io 등 오픈소스 생태계를 겨냥한 공급망 공격(Supply Chain Attack)이 정교화되고 있으며, GitHub 버전 태그를 악용한 credential-stealing malware 유포가 빈번합니다. 또한, 기존의 단순 피싱을 넘어 AI 기반의 자동화된 공격과 클라우드 환경(AWS GovCloud, M365)의 설정 오류를 노리는 위협이 기업 보안의 핵심 리스크로 부상했습니다.

2. 주요 위협 행위자 / 캠페인


- Lazarus Group: 금융 및 암호화폐 기업을 대상으로 RemotePE memory-only RAT을 배포하는 공격 수행.
- Webworm: Discord와 Microsoft Graph API를 활용하여 EU 정부 기관을 타겟팅하는 APT 캠페인 전개.
- Screening Serpens: AppDomainManager hijacking 기법을 사용하여 기술 및 방산 분야를 공격하는 이란계 APT 그룹.
- UNC6671: BlackFile vishing 및 extortion 캠페인을 통해 기업 대상 금전적 갈취 시도.
- ClickFix Campaign: Ghost CMS의 SQL injection 취약점을 악용하여 악성 JavaScript를 주입하는 대규모 캠페인.

3. 한국 관련 위협


- Nova 랜섬웨어: 국내 화장품 및 생활화학 기업을 대상으로 랜섬웨어 공격 수행 (AhnLab ASEC).
- 피싱 공격: 국세청 및 금융기관을 사칭하여 PhaaS(Phishing-as-a-Service)를 이용한 국내 기업 타겟형 피싱 메일 유포 (이스트시큐리티 알약).
- 금융권 위협: 국내 금융권 대상 1단계 피싱, 2단계 백도어/다운로더, 3단계 스틸러/랜섬웨어로 이어지는 다단계 공격 통계 확인.

4. 주요 CVE / 취약점 (CVSS 우선)


- CVE-2026-41940: cPanel, CVSS 9.8, Sorry 랜섬웨어에 의해 활발히 악용 중.
- CVE-2026-26980: Ghost CMS, SQL injection, ClickFix 캠페인에 의해 활발히 악용 중.
- CVE-2026-46725: TYPO3 CMS, CVSS 9.2, PHP object injection을 통한 RCE 위험.
- CVE-2026-5194: WolfSSL, CVSS 9.1, AI 기반 취약점 탐색으로 발견된 고위험 취약점.
- CVE-2026-46333: Linux Kernel, Local Root Privilege Escalation 및 Credential Disclosure 위험.

6. 즉각 대응 권고


- 공급망 보안 강화: npm, PyPI 등 오픈소스 패키지 사용 시 2FA-Gated Publishing 설정을 활성화하고, 의존성 패키지의 무결성을 정기적으로 검증하십시오.
- 클라우드 설정 점검: CISA의 SCuBA 프레임워크를 참조하여 AWS 및 M365 환경의 접근 권한 및 API 키 노출 여부를 즉시 감사하십시오.
- 취약점 패치: Ghost CMS(CVE-2026-26980) 및 cPanel(CVE-2026-41940) 사용 환경은 즉각적인 패치를 수행하고, 외부 노출된 관리 페이지의 접근 제어를 강화하십시오.
