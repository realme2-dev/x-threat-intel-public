1. 현재 주요 사이버보안 트렌드


최근 위협 환경은 공개된 취약점(CVE)이 발표된 지 13시간 이내에 즉각적으로 무기화되는 'Zero-Day/N-Day' 공격의 가속화가 두드러집니다. 특히 AI 모델을 활용한 취약점 탐색 및 공격 자동화 시도가 증가하고 있으며, 공급망 공격(Supply Chain Attack)을 통해 개발자 자격 증명을 탈취하는 정교한 캠페인이 지속되고 있습니다. 또한, 공격자들이 가정용 공유기나 IoT 기기를 공격 은폐를 위한 프록시 네트워크로 악용하는 사례가 국제적인 보안 경고의 핵심으로 부상했습니다.

2. 주요 위협 행위자 / 캠페인


- UNC6692: Microsoft Teams를 통해 IT Helpdesk로 위장하여 사용자에게 접근, 커스텀 악성코드 제품군을 배포하는 사회공학적 기법 사용 (Mandiant Blog).
- Tropic Trooper: 일본의 타겟과 가정용 공유기를 주요 공격 대상으로 삼으며, 도구와 TTP를 빠르게 변화시키는 중국 연계 APT 그룹 (Dark Reading).
- GopherWhisper: 몽골 정부 기관을 타겟으로 활동하는 신규 중국 연계 APT 그룹 (WeLiveSecurity).
- TeamPCP: Bitwarden CLI 및 NPM 패키지를 타겟으로 한 공급망 공격 캠페인 수행 (SecurityWeek).
- 0APT: 경쟁 랜섬웨어 조직인 Krybit의 내부 자료를 탈취하고 협박하는 범죄 조직 간 해킹전 양상 (데일리시큐).

3. 한국 관련 위협


- 기업뱅킹 소프트웨어(WGear) 취약점: 금융보안원 및 KISA를 통해 기업뱅킹 전자금융 서비스의 대용량 처리 프로그램 취약점 대응 공지 (이스트시큐리티 알약).
- 국내 APT 공격 동향: 2026년 3월 기준, 국내 타겟 대상 스피어 피싱 이메일 및 .lnk 파일 기반의 파워쉘 실행 공격이 주를 이룸 (AhnLab ASEC).

4. 주요 CVE / 취약점


- CVE-2026-3844: Breeze Cache WordPress Plugin, CVSS 9.8(Critical), 인증되지 않은 원격 파일 업로드 및 사이트 탈취 가능, 현재 악용 중.
- CVE-2026-21643: Fortinet FortiClient EMS 7.4.4, CVSS 9.8(Critical), 사전 인증 SQL 인젝션 및 RCE 가능, 현재 악용 중.
- CVE-2026-33626: LMDeploy, High-severity, LLM 배포 툴킷 SSRF 취약점, 공개 13시간 만에 악용 시작.
- CVE-2026-32201: Microsoft SharePoint Server, Actively exploited, CISA 긴급 패치 대상.
- CVE-2026-28950: Apple Notification Services, iOS/iPadOS 취약점, 삭제된 알림 데이터 복구 가능, 악용 사례 확인.

- 공급망 보안 강화: 개발 환경에서 사용 중인 NPM 패키지(특히 Bitwarden CLI 관련)의 무결성을 검증하고, 의심스러운 의존성 업데이트를 차단하십시오.
- 공유기 및 IoT 기기 보안: 기업 네트워크 내부의 가정용 공유기나 IoT 장비가 외부와 직접 통신하지 않도록 격리하고, 최신 펌웨어 업데이트를 적용하십시오.
