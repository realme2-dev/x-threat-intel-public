1. 현재 주요 사이버보안 트렌드


최근 공격자들은 LLM을 활용한 코드 생성 및 도메인 환각(Hallucination)을 이용한 공급망 공격, 에이전트 기반의 자동화된 침투 기술을 적극 도입하고 있습니다. 특히 Linux 커널 취약점(Bad Epoll)과 같은 로컬 권한 상승(LPE) 취약점이 지속적으로 발견되고 있으며, 사회공학적 기법과 결합된 PhaaS(Phishing-as-a-Service) 플랫폼이 기업 계정 탈취의 주요 경로로 자리 잡았습니다. 또한, 임베디드 장비 및 산업용 네트워크를 겨냥한 공격이 증가하며, AI 기반의 방어 체계와 공격자 간의 기술 격차가 더욱 심화되는 양상을 보입니다.

2. 주요 위협 행위자 / 캠페인


  • Armored Likho (Securelist): BusySnake Stealer를 사용하여 러시아, 브라질, 카자흐스탄의 정부 및 전력 부문을 타겟팅. AI 생성 로더와 LNK 파일 기반의 CVE-2025-9491 악용.

  • Scattered Spider (Krebs on Security): Transport for London 공격 관련 핵심 인물들이 체포 및 유죄 판결을 받으며 활동이 가시화됨.

  • TAG-182 (Recorded Future): MarkiRAT을 통해 이란 국내 타겟을 대상으로 사이버 감시 작전 수행.

  • TheSweetNight (CyberXlx9q): 인도네시아 국가사이버암호청(BSSN) 대상 해킹 주장.

  • World Leaks (Twitter/X): Tata Electronics를 타겟으로 한 랜섬웨어 공격 및 데이터 유출 수행.


    • 3. 한국 관련 위협


  • Tata Electronics 데이터 유출: 한국 대기업의 해외 법인 데이터가 World Leaks 그룹에 의해 유출된 정황이 보고됨.

  • 공급망 공격: 이스트시큐리티 ESRC는 북한 연계 해커가 유명 테스트 프레임워크인 Chai.js의 플러그인으로 위장한 악성 npm 패키지(chai-as-init)를 배포한 사례를 확인.

  • 기타: 국내 호텔 업계를 타겟으로 한 TONResolver 악성코드 유포 및 김수키(Kimsuky)의 스피어 피싱 공격이 지속적으로 관찰됨.


    • 4. 주요 CVE / 취약점


  • CVE-2026-46242 (Bad Epoll): Linux Kernel 6.4+ 대상, Unprivileged User가 Root 권한 획득 가능. (심각도: High/Critical)

  • CVE-2026-35280: Oracle WebCenter Enterprise Capture 내 심각한 취약점, 즉각적인 패치 권고.

  • CVE-2025-9491: Armored Likho가 BusySnake Stealer 배포 시 악용하는 취약점.

  • CVE-2026-56645 외 6건: Chromium-based Edge 브라우저 내 RCE 취약점 다수.

  • 공급망 보안 강화: 개발 환경에서 사용하는 npm 패키지(chai-as-init 등)의 무결성을 검증하고, 의심스러운 의존성 추가를 차단하십시오.
