1. 현재 주요 사이버보안 트렌드


최근 공격자들은 교육 플랫폼(Canvas)과 같은 대규모 공급망을 타겟팅하여 광범위한 데이터 유출 및 서비스 중단을 야기하고 있습니다. 특히 AI 모델을 활용한 취약점 탐색 및 자동화된 공격이 증가하고 있으며, Linux 커널 취약점(Dirty Frag, Copy Fail)을 악용한 LPE(Local Privilege Escalation) 공격이 빈번하게 발생하고 있습니다. 또한, 보안 도구(Checkmarx KICS 등)를 역이용한 공급망 공격과 클라우드 환경의 비밀 정보(Secrets) 탈취를 노리는 정교한 악성코드(PCPJack)가 주요 위협으로 부상했습니다.

2. 주요 위협 행위자 / 캠페인


  • ShinyHunters: Instructure(Canvas)를 대상으로 2차 공격을 감행하여 수천 개의 교육 기관 데이터 유출 및 로그인 포털 변조 수행.

  • ScarCruft: Yanbian 지역을 타겟으로 Windows/Android 게임 내 백도어를 삽입하는 공급망 공격 수행.

  • RansomHouse: Trellix 소스 코드 저장소 침해 및 데이터 유출 주장.

  • OceanLotus: PyPI 패키지를 통해 ZiChatBot 악성코드를 유포하는 캠페인 확인.

  • UNC6692: 사회 공학적 기법을 활용하여 커스텀 악성코드 제품군을 배포하는 다단계 침입 캠페인 수행.


    • 3. 한국 관련 위협


  • 자라(Zara) 코리아: 제3자 호스팅 데이터베이스 보안 사고로 인해 고객 정보 유출 발생(ShinyHunters 연루).

  • 기타: 한국어 트윗에서 특정 국가 기반의 공격 시도 및 AI 보안 관련 논의가 활발하나, 국내 인프라를 직접 타겟팅한 신규 대규모 공격은 보고되지 않음.


    • 4. 주요 CVE / 취약점


  • CVE-2026-6973: Ivanti EPMM, 원격 코드 실행(RCE), 제로데이 악용 확인.

  • CVE-2026-31431: Linux Kernel, LPE(Copy Fail), 심각도 높음.

  • CVE-2026-0300: PAN-OS, 버퍼 오버플로우(RCE), 제로데이 악용 확인.

  • CVE-2026-43284: Linux Kernel, LPE(Dirty Frag), 로컬 권한 상승.

  • CVE-2025-68670: xrdp, 사전 인증 RCE, 패치 권고.

  • Linux 커널 보안 패치: Copy Fail(CVE-2026-31431) 및 Dirty Frag(CVE-2026-43284) 대응을 위해 서버 커널 업데이트를 최우선으로 수행하십시오.
