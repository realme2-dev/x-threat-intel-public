1. 현재 주요 사이버보안 트렌드


최근 공격자들은 LLM과 AI 도구를 활용하여 공격의 전 과정을 자동화하는 'Autonomous Attack' 단계로 진입하고 있으며, 특히 JADEPUFFER와 같은 사례처럼 정찰부터 데이터 파괴까지 31초 만에 수행하는 자율형 공격이 보고되고 있습니다 (connect24h). 또한, macOS 환경을 타겟으로 한 CrashStealer와 같이 Notarized Dropper를 이용해 Gatekeeper를 우회하는 정교한 정보 탈취형 악성코드가 확산 중입니다 (TheHackerNews). 공급망 공격 역시 지속되어 Jscrambler npm 패키지 백도어 사례와 같이 신뢰받는 개발 도구를 악용한 침해 사고가 빈번하게 발생하고 있습니다 (BleepingComputer).

2. 주요 위협 행위자 / 캠페인


- Static Tundra (FSB Center 16): 통신, 에너지, 금융, 정부 등 6개 핵심 산업의 네트워크 장비를 타겟으로 설정 오류 및 취약점을 악용하여 침투 (CyberScoop).
- Cavern Manticore: 이란 연계 APT 그룹으로, 이스라엘을 주요 타겟으로 하는 모듈형 C2 프레임워크를 운영 중 (Check Point Research).
- Armored Likho: 러시아 내 조직을 대상으로 spear-phishing과 AI 생성 로더, BusySnake Stealer를 활용한 캠페인 수행 (Securelist).
- The Gentlemen: 어필리에이트 모델을 기반으로 급성장 중인 랜섬웨어 그룹으로, 기업 대상 파괴적 공격 수행 (Unit42).

3. 한국 관련 위협


- 일본 교통(Nihon Kotsu) 사이버 공격: 일본 최대 택시 운영사의 시스템이 마비되었으며, 한국 내 보안 커뮤니티에서도 해당 사건을 공급망 및 인프라 위협 사례로 주시 중 (securityLab_jp).
- 피싱 메일 유포: 프로젝트 제안서 및 송금 확인서를 위장한 피싱 메일이 국내 기업을 대상으로 지속 유포 중 (AhnLab ASEC).
- 북한 연계 공격: chai-as-init npm 패키지 공급망 공격 및 Kimsuky의 스피어 피싱 공격이 지속적으로 확인됨 (이스트시큐리티 알약).

4. 주요 CVE / 취약점


- CVE-2026-59801: 9Router ~0.4.41, CVSS 9.8, 인증되지 않은 원격 코드 실행 취약점 (OrizonCyber).
- CVE-2026-0288: PAN-OS 10.2~12.1, CVSS 9.2, User-ID TSA 설정 시 원격 코드 실행 가능 (connect24h).
- CVE-2026-5721: RabbitMQ, CVSS 8.7, OAuth client secret 노출로 인한 브로커 탈취 가능 (SecurityWeek).
- CVE-2026-62186: OpenClaw < 2026.6.8, CVSS 7.6, HTTP 모델 오버라이드 권한 우회 (KaitanSecurity).
