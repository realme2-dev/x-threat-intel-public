1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 모델(Mythos 등)을 활용하여 취약점 탐색 및 공격 코드 생성을 가속화하고 있으며, 이는 방어자가 대응해야 할 속도와 규모를 압도하고 있습니다. 특히 Microsoft Defender와 같은 보안 솔루션의 취약점을 직접 타겟팅하여 권한 상승 및 방어 체계 무력화를 시도하는 사례가 증가하고 있습니다. 또한, 공급망 공격(npm)과 더불어 AI 에이전트의 ID 관리 및 거버넌스가 새로운 보안 과제로 부상하고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • UNC6671 (Mandiant): Vishing을 결합한 BlackFile 랜섬웨어 및 갈취 캠페인 수행.

  • UNC6692 (Mandiant): 사회공학적 기법을 활용한 커스텀 악성코드 스위트 배포.

  • Webworm (ESET): 새로운 burrowing 기술을 추가하여 APT 공격 지속.

  • Kimwolf (Krebs on Security): IoT 봇넷을 구축하여 수백만 대의 기기를 감염시킨 혐의로 운영자 'Dort' 체포.


    • 3. 한국 관련 위협


  • Nova 랜섬웨어: 대한민국 화장품 및 생활화학 기업 대상 공격 확인 (AhnLab ASEC).

  • 텔레그램 스미싱: 텔레그램 보안 문제를 사칭하여 로그인 코드 및 전화번호를 탈취하는 캠페인 재확산 (AhnLab ASEC).

  • 피싱 메일: 국세청 및 하나은행을 사칭하여 PhaaS(Phishing-as-a-Service)를 이용한 기업 타겟 공격 포착 (이스트시큐리티 알약).


    • 4. 주요 CVE / 취약점


  • CVE-2026-20223: Cisco Secure Workload, CVSS 10.0(Critical), REST API 인증 우회 및 Site Admin 권한 획득 가능.

  • CVE-2026-41091: Microsoft Defender, CVSS 7.8, 로컬 권한 상승(SYSTEM), 실제 악용 중.

  • CVE-2026-46333: Linux Kernel (ptrace), 로컬 루트 권한 상승 및 기밀 파일 유출 가능.

  • CVE-2026-34926: TrendAI Apex One, 경로 탐색(Path Traversal), 실제 악용 중.

  • CVE-2026-9082: Drupal Core, SQL Injection, Highly Critical, PostgreSQL 환경에 영향.

  • Cisco Secure Workload 패치: REST API 취약점(CVE-2026-20223)에 대한 벤더 권고안에 따라 즉시 패치를 수행하십시오.

  • 텔레그램 보안 강화: 임직원 대상 텔레그램 로그인 코드 탈취 스미싱 주의보를 발령하고, 2단계 인증(2FA) 설정을 강제하십시오.
