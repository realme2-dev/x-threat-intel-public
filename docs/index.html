1. 현재 주요 사이버보안 트렌드


최근 공격자들은 Microsoft Teams와 같은 협업 도구를 활용한 사회공학적 기법과, JavaScript 런타임(Bun) 등 합법적인 개발 도구를 악용하여 탐지를 회피하는 정교한 공급망 공격을 강화하고 있습니다. 특히 교육 및 공공 부문을 타깃으로 한 대규모 데이터 유출 사건이 빈번하며, Palo Alto PAN-OS와 같은 핵심 네트워크 장비의 Zero-day 취약점을 악용한 원격 코드 실행(RCE) 공격이 활발합니다. 또한, AI 모델을 활용한 취약점 탐색 및 자동화된 공격이 현실화되면서 방어 체계의 패러다임 전환이 요구되고 있습니다.

2. 주요 위협 행위자 / 캠페인


- ShinyHunters: Instructure(Canvas) 및 한국 국방부 DB 침해 주장, 전 세계 교육기관 데이터 탈취.
- MuddyWater: Microsoft Teams를 악용한 자격 증명 탈취 및 False Flag 랜섬웨어 공격 수행.
- ScarCruft: Windows/Android 게임 플랫폼을 통한 공급망 공격 및 백도어 유포.
- OceanLotus: PyPI 패키지에 ZiChatBot 악성코드를 삽입하여 Windows/Linux 타깃 공격.
- xlabs_v1: ADB(Android Debug Bridge) 노출 IoT 기기를 감염시켜 DDoS 봇넷으로 활용.

3. 한국 관련 위협


- 국방부 데이터 유출: ShinyHunters가 BreachForums에 대한민국 국방부 DB 침해를 주장하며 데이터 유출 게시글을 업로드함.
- 교육기관 피해: Instructure(Canvas) 해킹으로 인해 국내 대학을 포함한 교육기관의 데이터 유출 가능성이 제기되어 점검이 시급함.
- 기업뱅킹 보안: 기업뱅킹 전자금융 소프트웨어(WGear) 취약점 관련 보안 주의보 발령.
