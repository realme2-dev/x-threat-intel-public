1. 현재 주요 사이버보안 트렌드


최근 사이버보안 환경은 AI 모델(Anthropic Mythos 등)을 활용한 대규모 취약점 탐색과 자동화된 공격이 급증하며 방어 체계의 패러다임 전환을 요구하고 있습니다. 특히 npm 및 Packagist와 같은 오픈소스 생태계를 겨냥한 공급망 공격(Supply Chain Attack)이 정교해지고 있으며, GitHub 저장소 내 자격 증명 유출 및 악성 코드 삽입이 빈번하게 발생하고 있습니다. 또한, 기존의 단순 피싱을 넘어 Microsoft 365 토큰 탈취 등 세션 하이재킹을 노리는 고도화된 피싱 캠페인이 기업 인프라를 위협하는 주요 벡터로 자리 잡았습니다.

2. 주요 위협 행위자 / 캠페인


  • Webworm (DarkReading/WeLiveSecurity): Discord 및 Microsoft Graph를 악용하여 EU 정부 기관을 타겟으로 하는 지속적인 정보 탈취 및 정찰 활동 수행.

  • UNC6671 (Mandiant): 'BlackFile' 캠페인을 통해 Vishing(음성 피싱)과 결합된 정교한 갈취 작전 수행.

  • Screening Serpens (Unit42): AppDomainManager 하이재킹 및 신규 RAT 변종을 사용하여 기술 및 방산 분야 타겟팅.

  • Anubis (Twitter/X): Signature Healthcare를 대상으로 2TB의 환자 데이터를 탈취하는 등 랜섬웨어 암호화 없이 데이터 유출 위주의 공격 수행.

  • 404Crew (Twitter/X): 이스라엘 기업 및 인프라를 대상으로 한 해킹 및 데이터 유출 활동 수행.


    • 3. 한국 관련 위협


  • Nova 랜섬웨어: 국내 화장품 및 생활화학 기업을 대상으로 한 랜섬웨어 공격 사례 확인 (AhnLab ASEC).

  • 피싱 메일: 국세청 및 하나은행을 정교하게 사칭하여 기업 계정 정보를 탈취하는 PhaaS(Phishing-as-a-Service) 기반 공격 포착 (이스트시큐리티 알약).


    • 4. 주요 CVE / 취약점


  • CVE-2026-20223: Cisco Secure Workload, CVSS 10.0 (Critical), 인증되지 않은 원격 공격자의 REST API 악용 가능.

  • CVE-2026-9082: Drupal, SQL Injection 취약점, 현재 전 세계적으로 활발히 악용 중.

  • CVE-2026-46529: Xreader/Evince/Atril, CVSS 8.4, 10년 이상 방치된 PDF 처리 관련 메모리 취약점.

  • CVE-2026-3102: ExifTool, macOS 시스템 타겟 이미지 기반 원격 코드 실행 취약점.

  • CVE-2026-46333: Linux Kernel, ptrace 관련 로컬 권한 상승 및 정보 유출 취약점.
