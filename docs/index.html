1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI를 활용한 공격의 고도화와 자동화가 핵심 트렌드로 자리 잡았습니다. 특히 Hugging Face와 같은 AI 모델 저장소가 자율형 AI 에이전트에 의해 해킹되는 사례가 보고되었으며, 공격자들은 AI 모델의 가드레일을 우회하거나 악성 코드를 생성하는 데 AI를 적극적으로 활용하고 있습니다(SecurityWeek). 또한, 소프트웨어 공급망 공격과 더불어 취약한 UEFI shim을 악용한 Secure Boot 우회 등 인프라 레벨의 지속적인 위협이 관찰되고 있습니다(WeLiveSecurity). 마지막으로, 기업 내부 채용 프로세스에 침투한 위협 행위자가 악성 코드를 삽입하는 공급망 침해 사례가 지속적으로 발생하고 있어 개발 환경의 보안 검증이 시급합니다(BleepingComputer).

2. 주요 위협 행위자 / 캠페인


- Lazarus Group: MetaMask 개발자로 위장 취업하여 암호화폐 관련 코드에 악성 행위를 시도한 정황이 포착됨(BleepingComputer).
- Cavern Manticore: 이란 연계 APT 그룹으로, 이스라엘을 타겟으로 하는 모듈형 C2 프레임워크를 운영 중(Check Point Research).
- Scattered Spider: 영국 Transport for London 공격 등 대규모 사이버 공격을 주도한 핵심 인물들이 징역형을 선고받음(CyberScoop).
- Inc Ransomware: SonicWall SMA 장비의 제로데이 취약점을 연쇄적으로 악용하여 루트 권한을 획득하는 공격 수행(DarkReading).

3. 한국 관련 위협


- 해양수산부: AI 응용제품 신속상용화 지원사업 착수 (데일리시큐)
- 이스트시큐리티 ESRC: 국세청 사칭 피싱 메일을 통한 원격제어 악성코드 유포 및 북한발 npm 악성 패키지('chai-as-init') 유포 사례 보고 (이스트시큐리티 알약, 이스트시큐리티 알약)
- AhnLab ASEC: 리눅스 SSH 서버 대상 코인 마이너 유포 및 인포스틸러 동향 보고 (AhnLab ASEC, AhnLab ASEC)

4. 주요 CVE / 취약점


- CVE-2026-6875: ServiceNow AI Platform 코드 실행 취약점, 현재 실제 공격에 악용 중 (BleepingComputer)
- CVE-2026-14266: 7-Zip XZ 아카이브 처리 시 발생하는 Heap-based buffer overflow, 코드 실행 가능 (TheHackerNews)
