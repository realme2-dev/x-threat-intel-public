1. 현재 주요 사이버보안 트렌드


최근 위협 환경은 Linux 커널 취약점(Dirty Frag, Copy Fail)을 악용한 Local Privilege Escalation(LPE) 공격이 급증하며 시스템 전반의 권한 탈취 위험이 커지고 있습니다. 또한, ShinyHunters와 같은 위협 행위자가 교육 플랫폼(Canvas)을 대상으로 대규모 데이터 유출 및 협박을 지속하고 있으며, 공급망 공격을 통해 신뢰받는 소프트웨어(JDownloader)의 설치 파일을 변조하는 방식이 활발합니다. AI 모델을 활용한 취약점 탐색 및 공격 자동화가 가속화됨에 따라, 보안 운영팀은 MITRE ATT&CK 프레임워크 기반의 대응 체계 고도화가 시급한 상황입니다.

2. 주요 위협 행위자 / 캠페인


- ShinyHunters: Instructure(Canvas)를 대상으로 2차 공격을 감행하여 수천 개 교육 기관의 학생 데이터 유출 및 로그인 포털 디페이스(Defacement) 수행.
- ScarCruft: Yanbian 지역을 타겟으로 백도어가 포함된 Windows/Android 게임을 유포하는 공급망 공격 수행.
- OceanLotus: PyPI 패키지 저장소에 ZiChatBot 악성코드를 포함한 패키지를 업로드하여 Windows/Linux 환경 타겟팅.
- UNC6692: 사회 공학적 기법(Social Engineering)을 활용하여 맞춤형 악성코드 스위트를 배포하는 다단계 침해 캠페인 수행.

3. 한국 관련 위협


- 이스트시큐리티 ESRC: 2026년 1분기 랜섬웨어 동향 보고서를 통해 국내 기업 대상 랜섬웨어 피해 사례 및 Wildcard DNS 기법을 악용한 피싱 메일 유포 주의 권고.
- 기타: 한국어 트윗 샘플 내에서 특정 한국 기관에 대한 직접적인 공격 언급은 없으나, 글로벌 교육 플랫폼(Canvas) 해킹으로 인한 국내 사용자들의 간접적인 영향 가능성이 제기됨.

4. 주요 CVE / 취약점


- CVE-2026-31431 (Copy Fail): Linux Kernel LPE, 심각도 Critical, 실제 악용 확인됨.
- CVE-2026-0300: PAN-OS Captive Portal RCE, 심각도 Critical, 인증 없이 원격 코드 실행 가능.
- CVE-2026-6973: Ivanti EPMM RCE, 심각도 High, 제로데이로 악용 확인됨.
- CVE-2026-29201: cPanel/WHM 권한 상승 및 코드 실행 취약점, 심각도 High.
- CVE-2025-68670: xrdp Pre-auth RCE, 심각도 High.
