1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI를 활용하여 탐지를 우회하는 기법(Prompt Injection, Comment Stuffing)을 고도화하고 있으며, 특히 공급망 공격을 통해 신뢰받는 패키지나 라이브러리를 오염시키는 사례가 빈번합니다(BleepingComputer). 또한, 기업의 클라우드 인프라와 API를 겨냥한 대규모 정찰 활동이 활발하며, 랜섬웨어 조직은 내부자(협상가 등)를 포섭하거나 악성 로더를 서비스 형태로 제공하는 등 운영 모델을 다각화하고 있습니다(SecurityWeek). 마지막으로, CISA BOD 26-04와 같은 규제에 대응하기 위해 3일 이내의 신속한 취약점 패치가 보안 운영의 핵심 과제로 부상했습니다(Qualys Blog).

2. 주요 위협 행위자 / 캠페인


- Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘 등 중동 지역을 타겟팅함(Check Point Research).
- Armored Likho: 러시아 타겟의 APT 캠페인으로, AI 생성 로더와 파이썬 기반의 BusySnake Stealer를 활용함(Securelist).
- BlackCat/ALPHV: 랜섬웨어 조직으로, 내부 협상가를 포섭하여 기업 정보를 탈취하고 협박하는 전술을 구사함(데일리시큐).
- ShinyHunters: 북미/유럽 지역 의료, 보안, 유통 분야를 대상으로 대규모 데이터 유출을 주도함(AhnLab ASEC).

3. 한국 관련 위협


- 한국 군사 사이버 공격: 한국 군 당국이 작년 18,000건 이상의 사이버 공격 시도를 방어했으며, 전문 인력 확보에 어려움을 겪고 있다는 보도(TheKoreaHerald).
- Kimsuky 스피어 피싱: 보안 담당자를 대상으로 개인정보 유출을 사칭한 스피어 피싱 공격 사례(이스트시큐리티 알약).
- 공급망 공격: 북한 연계 해커가 유명 npm 패키지(chai.js) 플러그인으로 위장한 악성 패키지 배포(이스트시큐리티 알약).

4. 주요 CVE / 취약점


- CVE-2026-20896: Gitea Docker, CVSS 9.8 (Critical), 인증 우회 및 사용자 사칭 가능(KasperskyDev).
- CVE-2026-53359 (Januscape): Linux KVM, Guest-to-Host 탈출 취약점, Intel/AMD 시스템 영향(IntCyberDigest).
- CVE-2026-47291: Windows HTTP.sys, RCE 취약점, PoC 공개됨(cyber_advising).
- CVE-2026-46215: Linux Kernel, Use-After-Free, Linux 7.0 버전 영향(bluedragonsec).
- CVE-2026-14126: Chrome for Android, 중요 취약점, 즉시 업데이트 권고(windowsforum).
