1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 모델을 활용해 취약점 탐색 속도를 높이고 있으며, 공급망 공격(Supply Chain Attack)과 LOLBIN(Living-off-the-Land Binaries)을 악용한 은밀한 침투가 급증하고 있습니다. 특히 OAuth Consent를 악용하여 MFA를 우회하거나, 합법적인 소프트웨어 서명 서비스를 악용하는 Malware-Signing-as-a-Service(MSaaS) 형태의 공격이 기업 보안 체계를 위협하고 있습니다. 또한, Verizon DBIR 보고서에 따르면 초기 침해의 31%가 취약점 익스플로잇에서 시작되고 있어, 패치 관리의 중요성이 그 어느 때보다 강조되는 시점입니다.

2. 주요 위협 행위자 / 캠페인


  • Fox Tempest: 합법적인 Microsoft Artifact Signing 서비스를 악용하여 랜섬웨어 및 악성코드를 서명·유포하는 MSaaS 운영.

  • UNC6671: 'BlackFile'이라는 명칭의 Vishing(음성 피싱) 기반 갈취 캠페인을 수행 중.

  • Kimsuky: PebbleDash 기반 도구를 사용하여 AppleSeed 악성코드 클러스터와 연계된 사이버 간첩 활동 수행.

  • FrostyNeighbor: 지속적인 사이버 첩보 활동을 위해 침해 체인을 업데이트하며 공격 수행.

  • TeamPCP: Jenkins 플러그인 취약점 등을 악용한 공급망 공격 캠페인 지속.


    • 3. 한국 관련 위협


  • 이스트시큐리티 ESRC: 국세청 및 하나은행을 사칭한 PhaaS 기반의 기업 타겟 피싱 메일 공격 포착.

  • ASEC: 국내 의료 초음파 장비 제조 기업의 데이터 유출 주장 및 4월 인포스틸러/APT 그룹 동향 보고서 발행.

  • 정부 대응: 과기정통부, 10월 개정 정보통신망법 시행을 앞두고 '침해사고 조사 심의위원회' 사전 가동.


    • 4. 주요 CVE / 취약점


  • CVE-2026-0300: PAN-OS Captive Portal, Buffer Overflow로 인한 RCE, 악용 중.

  • CVE-2026-31635: Linux Kernel, LPE(Local Privilege Escalation), PoC 공개됨.

  • CVE-2026-4293: Kieback & Peter DDC Building Controllers, 브라우저 제어권 탈취 가능.

  • CVE-2026-41957: F5 BIG-IP/BIG-IQ, 인증된 사용자 대상 Deserialization RCE.

  • CVE-2025-51427: ModelScope 1.25.0, 설정 파일 조작을 통한 Arbitrary Code Execution.

    6. 즉각 대응 권고


  • MSHTA 및 LOLBIN 모니터링: MSHTA 유틸리티를 통한 비정상적인 프로세스 실행 여부를 EDR로 탐지하고, 불필요한 경우 실행 제한 정책 적용.

  • OAuth 권한 검토: Microsoft 365 환경에서 사용자 동의(Consent) 기반의 MFA 우회 공격을 방지하기 위해 OAuth 앱 권한을 전수 조사하고 과도한 권한 제거.

  • 패치 및 취약점 관리: PAN-OS 및 Linux Kernel 등 공개된 PoC가 존재하는 취약점에 대해 즉각적인 패치 적용 및 노출된 관리 인터페이스의 외부 접근 차단.
