1. 현재 주요 사이버보안 트렌드


최근 공격자들은 LLM 기반의 AI 에이전트를 활용하여 취약점 탐색 및 사후 공격(post-exploitation)을 자동화하는 추세입니다. 특히 공급망 공격이 npm 저장소 등을 통해 지속적으로 발생하고 있으며, 공개된 취약점(CVE)에 대한 PoC가 나오자마자 즉각적인 악용(Active Exploitation)으로 이어지는 주기가 매우 짧아졌습니다. 또한, Windows Netlogon과 같은 핵심 인프라 취약점이 실무 환경에서 직접적인 RCE 공격에 활용되고 있어 패치 적용의 시급성이 극대화되었습니다.

2. 주요 위협 행위자 / 캠페인


  • SideCopy: 아프가니스탄 재무부를 타겟으로 XenoRAT을 배포하여 지속적인 감시 및 정보 탈취 수행.

  • UNC6671: BlackFile 캠페인을 통해 Vishing(음성 피싱)과 결합된 대규모 갈취 작전 수행.

  • Screening Serpens: AppDomainManager hijacking 기법을 사용하여 기술 및 방산 분야를 타겟으로 한 첩보 활동 전개.

  • Kimsuky: 국내 카드사 보안 메일로 위장한 악성 LNK 파일을 통해 악성코드 유포.


    • 3. 한국 관련 위협


  • 카드사 사칭 악성 메일: 안랩 ASEC에 따르면, Kimsuky 그룹의 수법과 유사한 악성 LNK 파일이 국내 카드사 보안 메일로 위장하여 유포 중입니다.

  • WP Maps Pro 취약점: 국내 사이트에서도 해당 플러그인의 CVE-2026-8732 취약점을 악용한 관리자 권한 탈취 시도가 확인되었습니다.

  • 공급망 위협: npm 저장소 내 Web3 및 보안 도구로 위장한 악성 패키지 유포가 지속적으로 관찰되고 있습니다.


    • 4. 주요 CVE / 취약점


  • CVE-2026-8732: WP Maps Pro, CVSS 9.8, 관리자 권한 탈취를 위한 Active Exploitation 진행 중.

  • CVE-2026-41089: Windows Netlogon, CVSS 미상(Critical), 인증 없는 RCE 및 SYSTEM 권한 획득 가능, Active Exploitation 중.

  • CVE-2026-45668: Trilium, CVSS 9.3, 악성 노트 파일 임포트를 통한 RCE 발생.

  • CVE-2026-46333: Linux Kernel, CVSS 미상, 로컬 권한 상승 및 Credential Disclosure.

  • CVE-2026-45247: Mirasvit Cache Warmer, CVSS 미상, 인증 없는 RCE 가능.

    6. 즉각 대응 권고


  • WP Maps Pro 패치: 해당 플러그인을 사용하는 모든 웹사이트는 즉시 최신 버전으로 업데이트하거나 취약점 패치를 적용하십시오.

  • Windows Netlogon 점검: 도메인 컨트롤러(DC)의 Netlogon 서비스 관련 보안 업데이트를 즉시 확인하고, 비정상적인 네트워크 트래픽을 모니터링하십시오.

  • 공급망 보안 강화: 개발 환경에서 npm 패키지 설치 시 출처를 검증하고, 자동화된 보안 스캔 도구를 통해 악성 패키지 유입을 차단하십시오.
