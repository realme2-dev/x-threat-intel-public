1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 코딩 에이전트와 LLM을 활용하여 취약점을 탐색하거나, 정상적인 개발 도구의 동작을 모방해 보안 솔루션의 탐지를 우회하는 고도화된 전술을 구사하고 있습니다(데일리시큐). 특히 공급망 공격의 일환으로 GitHub 저장소나 npm 패키지를 오염시키는 사례가 빈번하며, AI가 생성한 환각(Hallucination) 도메인을 악용한 'Phantom Squatting'이 새로운 위협 벡터로 부상했습니다(Unit42). 또한, 기존의 단순 피싱을 넘어 vishing, device code phishing, MFA abuse를 결합한 ID 중심의 데이터 탈취 공격이 증가하고 있습니다(BleepingComputer).

2. 주요 위협 행위자 / 캠페인


- Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘을 포함한 타겟을 공격 중(Check Point Research).
- TAG-182: 이란 연계 클러스터로, MarkiRAT 악성코드를 사용하여 VPN/미디어 앱으로 위장한 사이버 감시 작전 수행(Recorded Future).
- Armored Likho: 러시아 타겟의 조직을 대상으로 BusySnake Stealer를 배포하며 AI 생성 로더를 활용하는 캠페인 수행(Securelist).
- Helix: SharePoint 데이터 탈취를 목적으로 vishing 및 MFA abuse 전술을 사용하는 신규 데이터 갈취 그룹(BleepingComputer).
