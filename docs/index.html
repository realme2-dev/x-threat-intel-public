1. 현재 주요 사이버보안 트렌드


최근 사이버 위협은 AI를 활용한 공격 자동화와 공급망 공격(Supply Chain Attack)이 결합된 형태로 진화하고 있습니다. 특히, 악성 npm 패키지 유포나 소프트웨어 업데이트 메커니즘을 악용한 침투가 빈번하며, 기업의 개발자 채용 프로세스를 우회하여 악의적인 행위자를 내부자로 침투시키는 사례가 보고되었습니다(BleepingComputer). 또한, SonicWall SMA와 같은 VPN 장비의 Zero-Day 취약점을 악용한 랜섬웨어 공격이 급증하고 있어, 경계 보안 장비의 패치 관리가 시급한 상황입니다(DarkReading).

2. 주요 위협 행위자 / 캠페인


- UAC-0145: ClickFix CAPTCHA 기법을 활용하여 우크라이나 대상 데이터 탈취형 악성코드 유포(TheHackerNews)
- Lazarus Group: 암호화폐 탈취 및 개발자 채용 위장 침투, npm 악성 패키지(chai-as-init) 유포 등 지속적인 공급망 공격(Securelist, 이스트시큐리티)
- Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘 등 타겟 공격(Check Point Research)
- Inc Ransomware: SonicWall SMA Zero-Day 취약점을 악용하여 기업 네트워크에 침투 및 랜섬웨어 배포(DarkReading)
- GoSerpent: 동남아시아 정부 기관을 타겟으로 하는 데이터 수집 및 유출 목적의 백도어 캠페인(Securelist)

3. 한국 관련 위협


- 공급망 위협: 북한발 악성 npm 패키지 'chai-as-init'이 유포되어 국내 개발 환경 및 공급망 보안에 주의가 요구됨(이스트시큐리티)
- 피싱 공격: 국세청 사칭 피싱 메일을 통해 원격제어 악성코드를 유포하는 사례 확인(이스트시큐리티)
- 기타: 한국 특정 위협은 아니나, 국내 보안 기업(AhnLab)이 리눅스 서버 대상 코인 마이너 유포 및 인포스틸러 동향을 지속 모니터링 중(AhnLab ASEC)

4. 주요 CVE / 취약점
