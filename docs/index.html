1. 현재 주요 사이버보안 트렌드


최근 사이버 보안 환경은 AI 기술을 악용한 공격의 고도화와 함께, Microsoft Patch Tuesday를 통해 기록적인 규모(622개)의 취약점이 공개되면서 패치 관리의 중요성이 극대화되었습니다 (TheHackerNews). 특히 공격자들은 정상적인 기업용 SaaS 연동(Salesforce 등)이나 OAuth 토큰을 악용하여 로그인 기록을 우회하는 'Living-off-the-land' 방식의 공격을 선호하고 있습니다 (데일리시큐). 또한, 생성형 AI를 활용한 피싱 메일 제작 및 악성코드 로더 생성 사례가 급증하며 방어 체계의 근본적인 변화를 요구하고 있습니다 (AhnLab ASEC).

2. 주요 위협 행위자 / 캠페인


  • Cavern Manticore: 이란 연계 APT 그룹으로, 모듈형 C2 프레임워크를 사용하여 이스라엘을 주요 타겟으로 활동 중 (Check Point Research).

  • Armored Likho: 러시아 지역 조직을 타겟으로 AI 생성 로더와 Python 기반의 'BusySnake Stealer'를 사용하는 APT 캠페인 수행 (Securelist).

  • The Gentlemen: 제휴 모델(Affiliate model)을 통해 급성장 중인 랜섬웨어 그룹으로, 기업 내부망 침투 및 데이터 탈취 주력 (Unit42).


    • 3. 한국 관련 위협


  • 피싱 메일 유포: 프로젝트 제안서 및 송금 확인서를 위장한 피싱 메일이 국내 기업을 대상으로 유포 중 (AhnLab ASEC).

  • 세무 위반 사칭: 국세 당국을 사칭하여 원격제어 악성코드를 유포하는 피싱 메일 확인 (이스트시큐리티 알약).

  • 공급망 공격: 과거 북한 연계 공격자가 npm 패키지('chai-as-init')를 위장하여 배포한 사례가 지속적으로 모니터링됨 (이스트시큐리티 알약).


    • 4. 주요 CVE / 취약점
