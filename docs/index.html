1. 현재 주요 사이버보안 트렌드


최근 공격자들은 AI 모델과 자동화된 워크플로우 플랫폼(Dify, Langflow 등)의 취약점을 악용하여 텐트 간 데이터 탈취 및 RCE를 시도하는 경향이 뚜렷합니다. 또한, 공급망 공격(Supply Chain Attack)이 WordPress 플러그인과 NPM 패키지 전반으로 확산되고 있으며, 인포스틸러(Infostealer)를 통해 수집된 대규모 계정 정보가 인터넷에 무방비로 노출되는 사례가 빈번합니다. 마지막으로, EDR을 무력화하는 프레임워크와 함께 합법적인 도구(ClickOnce, RMM)를 악용한 다단계 피싱 공격이 고도화되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • The Gentlemen: EDR Killer 프레임워크를 운용하는 RaaS 그룹으로, 공격적인 채용 전략을 통해 활동 범위를 빠르게 확장 중.

  • FishMonger: SprySOCKS for Windows 백도어를 사용하여 커널 드라이버를 악용하는 등 고도의 은닉 기술을 구사함.

  • North Korean Hackers (Kimsuky 등): Mastra NPM 패키지 공급망 공격 및 보안 담당자를 타겟으로 한 스피어 피싱 캠페인 수행.

  • Scattered Spider: Transport for London(TfL) 사이버 공격에 연루되어 법적 처벌 절차 진행 중.


    • 3. 한국 관련 위협


  • AryStinger 봇넷: 한국 내 구형 D-Link 공유기를 감염시켜 공격 중계지로 악용하는 사례가 다수 발견됨.

  • 공급망 공격: Chai.js 플러그인으로 위장한 악성 NPM 패키지(chai-as-init)가 국내 개발 환경을 타겟으로 배포됨.

  • 금융권 피싱: 2026년 5월 기준, 국내 금융권 대상 초기 침투 경로로 피싱(Phishing) 공격이 가장 높은 비중을 차지함.


    • 4. 주요 CVE / 취약점


  • CVE-2024-4577: PHP-CGI (Windows), CVSS 9.8, RCE 취약점, 악용 사례 다수.

  • CVE-2026-49468: LiteLLM, CVSS 9.5, 인증 우회, Exploit in the wild.

  • CVE-2026-8461: FFmpeg (PixelSmash), CVSS 8.8, Heap OOB Write, RCE 가능.

  • CVE-2026-47729: Squid Proxy (Squidbleed), CVSS 미상(High급), 정보 유출.

  • CVE-2026-55447: Langflow, 인증 토큰 탈취 및 RCE 가능.

  • 패치 및 업데이트: 사용 중인 FFmpeg, Squid Proxy, LiteLLM, Langflow 버전을 즉시 최신으로 업데이트하십시오.

  • 공유기 보안 점검: D-Link 등 구형 공유기 사용 환경을 전수 조사하고, 펌웨어 업데이트가 불가능한 기기는 네트워크 격리 또는 교체하십시오.

  • 공급망 보안 강화: NPM 패키지 의존성 관리 시 chai-as-init 등 의심스러운 패키지 포함 여부를 확인하고, 개발 파이프라인 내 시크릿 관리(GitHub Token 등) 정책을 재점검하십시오.
