1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI를 활용한 공격 자동화와 자율 에이전트(Autonomous AI Agent)를 결합한 공격이 급증하고 있으며, 특히 공격자가 모델 주변에 구축한 'scaffolding'이 공격의 성공 여부를 결정짓는 핵심 요소로 부상했습니다 (MalwareBibleJP). WordPress 환경에서는 CVE-2026-63030과 CVE-2026-60137을 연쇄적으로 활용한 'wp2shell' 공격이 대규모로 스캐닝되고 있어 웹 인프라의 즉각적인 패치가 요구됩니다 (TheHackerNews). 또한, Qilin 랜섬웨어 조직과 같이 기존의 VPN 취약점을 악용하여 기업 네트워크를 침투하는 랜섬웨어 캠페인이 활발하며, 공급망 공격 및 제로데이 취약점 악용이 지속적으로 관찰되고 있습니다 (BleepingComputer).

2. 주요 위협 행위자 / 캠페인


- JADEPUFFER: AI 에이전트 기반의 랜섬웨어 운영자로, Langflow 서버를 타겟으로 AI 모델 파일을 암호화하는 ENCFORGE 랜섬웨어 공격 수행 (TheHackerNews).
- Qilin: Palo Alto Networks GlobalProtect VPN의 인증 우회 취약점을 악용하여 기업 네트워크에 침투하는 랜섬웨어 캠페인 전개 (BleepingComputer).
- World Leaks: 인도 쿠단쿨람 원자력발전소 건설 협력사를 해킹하여 한국 기업 자료를 포함한 대규모 데이터를 유출함 (데일리시큐).
- Storm-2603: SharePoint RCE 취약점을 악용하여 Warlock 랜섬웨어를 유포하는 활동 포착 (infrasecserv).

3. 한국 관련 위협


- 인도 원전 협력사 해킹: 인도 쿠단쿨람 원자력발전소 건설 협력사 해킹으로 인해 국내 기업 및 기관 관련 파일 2,432건이 다크웹에 노출됨 (데일리시큐).
- 북한발 npm 악성 패키지: Chai.js 플러그인으로 위장한 악성 npm 패키지('chai-as-init') 유포 사례 확인 (이스트시큐리티 알약).

4. 주요 CVE / 취약점


- CVE-2026-53412: Zoom Windows 클라이언트, RCE/계정 탈취 가능, CVSS 9.8 (Critical), 악용 가능성 높음 (MalwareBibleJP).
- CVE-2026-63030 / CVE-2026-60137: WordPress Core, RCE, 대규모 스캐닝 및 악용 중 (TheHackerNews).
- CVE-2026-6875: ServiceNow AI Platform, RCE, 공개 직후 악용 사례 보고 (SecurityWeek).
- CVE-2026-0257: Palo Alto GlobalProtect VPN, 인증 우회, Qilin 랜섬웨어에 의해 악용 중 (BleepingComputer).
