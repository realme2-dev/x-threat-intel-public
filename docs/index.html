1. 현재 주요 사이버보안 트렌드


최근 공격자들은 단순한 소프트웨어 취약점 악용을 넘어, 관리자의 계정 탈취 및 사회공학적 기법을 결합한 정교한 공급망 공격(Supply Chain Attack)에 집중하고 있습니다. 특히 Axios npm 패키지 사례와 같이 개발 환경을 직접 타겟팅하여 신뢰받는 라이브러리를 오염시키는 방식이 확산되고 있습니다. 또한, AI 에이전트 및 멀티 에이전트 환경에서의 프롬프트 인젝션과 같은 새로운 공격 표면이 등장했으며, Linux 서버 내 PHP 웹쉘을 HTTP cookie로 제어하는 은밀한 지속성 유지 기법이 관찰되고 있습니다.

2. 주요 위협 행위자 / 캠페인


- TeamPCP: CERT-EU가 확인한 유럽연합 기관 대상 클라우드 침해 주체로, Vect 랜섬웨어 그룹과 파트너십을 맺고 공급망 공격을 수행 중.
- TA416: PlugX 및 OAuth 기반 피싱을 사용하여 유럽 정부 및 외교 기관을 타겟팅하는 중국 연계 그룹.
- Kimsuky: 한국 대상 이력서 및 대북 정책 문서 위장 LNK 파일 유포, 최근 파이썬 기반 백도어 실행 구조 변경.
- UNC1069: Axios npm 패키지 유지보수 담당자를 사회공학적으로 공격하여 공급망 침해를 유도한 북한 연계 그룹.
- Qilin: 독일 정당 Die Linke를 공격하여 데이터를 탈취하고 시스템 중단을 야기한 랜섬웨어 그룹.
