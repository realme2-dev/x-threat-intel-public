1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 AI를 활용한 공격 기법의 고도화와 공급망 공격의 확산이 두드러집니다. 공격자들은 LLM을 활용해 취약점을 탐색하거나 공격 코드를 생성하고 있으며, 특히 FakeGit 캠페인과 같이 대규모 GitHub 저장소를 악용하여 SmartLoader 및 StealC를 유포하는 등 자동화된 공격이 증가하고 있습니다(BleepingComputer). 또한, wp2shell 취약점과 같이 공개된 취약점에 대해 패치 배포 직후 공격이 시작되는 'Zero-Day' 및 'N-Day' 악용 속도가 매우 빨라지고 있어 신속한 패치 관리의 중요성이 강조됩니다(Malwarebytes Labs).

2. 주요 위협 행위자 / 캠페인


- Storm-2603: CVE-2026-50522를 악용하여 Microsoft SharePoint 서버에서 머신 키를 탈취하고 Warlock 랜섬웨어를 배포함 (Twitter/infrasecserv)
- FakeGit Campaign: 7,600개의 GitHub 저장소를 통해 SmartLoader와 StealC 인포스틸러를 유포 (BleepingComputer)
- Anubis Ransomware: Coca-Cola의 자회사 Fairlife를 공격하여 데이터 유출 협박 (BleepingComputer)
- Trim: AI 모델의 Jailbreak를 통해 공격용 플랫폼을 구축하고 보안 도구와 통합함 (DarkReading)

3. 한국 관련 위협


- 국립외교원 해킹: 온라인교육시스템이 제로데이 취약점 등으로 10개월간 장악되어 외교관 및 재외공관 인력 정보 최대 1만 건 노출 (데일리시큐)
- 인도 원전 해킹 연루: 인도 쿠단쿨람 원전 협력사 해킹으로 인해 국내 기업 및 기관 관련 자료 2,432건이 다크웹에 유출 (데일리시큐)
- 북한발 npm 악성 패키지: chai-as-init 패키지를 통해 공급망 공격 시도 (이스트시큐리티 알약)
