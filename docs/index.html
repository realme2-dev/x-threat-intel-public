1. 현재 주요 사이버보안 트렌드


최근 사이버 위협은 AI 모델 자체의 취약점(Model Hijacking, Prompt Injection)과 AI를 활용한 공격 자동화가 결합되어 공격 속도가 비약적으로 빨라지고 있습니다. 특히 공급망 공격(Supply Chain Attack)을 통해 OAuth Token을 탈취하거나, 레거시 자격 증명을 악용하여 엔터프라이즈 통합 인프라를 침투하는 사례가 급증하고 있습니다. 또한, 공격자들이 AI 분석을 회피하기 위해 악성코드 내부에 무관한 텍스트를 삽입하는 등 방어 체계를 우회하려는 시도가 관찰됩니다.

2. 주요 위협 행위자 / 캠페인


  • Gentlemen (WeLiveSecurity): EDR killer framework를 활용하여 보안 솔루션을 무력화하는 RaaS 그룹으로, 현재 가장 활발한 랜섬웨어 공격자 중 하나임.

  • Kimsuky (AhnLab ASEC): 보안 담당자를 타겟으로 한 스피어 피싱 공격을 지속하며, 개인정보 유출을 미끼로 악성 첨부파일 실행을 유도함.

  • FishMonger (WeLiveSecurity): SprySOCKS for Windows 백도어를 통해 커널 드라이버를 악용하는 고도화된 스텔스 공격 수행.

  • ShinyHunters (Malwarebytes): 대규모 데이터 유출(Kodak 등)을 주도하며, 탈취한 데이터를 기반으로 금전적 협박을 가하는 위협 행위자.

  • TeamPCP (CyberScoop): 오픈소스 소프트웨어 생태계를 타겟으로 하여 공급망 신뢰 모델을 파괴하는 대규모 공격 캠페인 수행.


    • 3. 한국 관련 위협


  • Kimsuky 스피어 피싱: 보안 담당자를 대상으로 한 정교한 피싱 공격이 지속적으로 탐지됨.

  • 공급망 위협: 북한 연계 해킹 그룹이 유명 테스트 프레임워크인 Chai.js의 플러그인으로 위장한 악성 npm 패키지(chai-as-init)를 배포한 사례 확인.

  • 금융권 피싱: 2026년 5월 기준 국내 금융권 대상 Phishing 공격이 최고치를 기록하며 초기 침투의 주요 경로로 활용됨.
