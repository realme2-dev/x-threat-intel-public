1. 현재 주요 사이버보안 트렌드


최근 보안 환경은 Supply Chain Attack의 정교화와 AI 기반 취약점 탐색 및 자동화 공격의 확산이 두드러집니다. 특히 Linux Kernel 및 Firewall 장비의 Zero-day 취약점이 실제 공격에 활발히 악용되고 있으며, 공격자들은 Cloud-based Phishing과 Credential Exfiltration을 결합하여 탐지를 우회하고 있습니다. 또한, 지정학적 갈등과 연계된 Critical Infrastructure 대상의 파괴적 공격 및 데이터 유출 시도가 지속적으로 증가하는 추세입니다.

2. 주요 위협 행위자 / 캠페인


  • ScarCruft (WeLiveSecurity): Yanbian 지역을 타겟으로 Windows 및 Android 게임 플랫폼에 백도어를 삽입하는 공급망 공격 수행.

  • Silver Fox (Securelist): 러시아 및 인도 기업을 대상으로 세무 당국을 사칭하여 ValleyRAT 및 ABCDoor 배포.

  • UNC6692 (Mandiant): 사회 공학적 기법(Social Engineering)을 활용하여 커스텀 악성코드 제품군을 배포하는 다단계 침입 캠페인 전개.

  • Scattered Spider (Krebs on Security): 기업 대상의 Wire Fraud 및 Identity Theft를 주도하며, 주요 멤버가 법적 처벌을 받는 등 활동이 가시화됨.


    • 3. 한국 관련 위협


  • APT 공격 동향: 안랩(ASEC) 보고서에 따르면, 국내 타겟 APT 공격은 주로 Spear Phishing 이메일을 통해 유포되며, .lnk 파일을 활용한 파워쉘 실행 방식이 가장 높은 비중을 차지함.

  • 기업뱅킹 취약점: 기업뱅킹 전자금융 소프트웨어 WGear의 보안 취약점 관련 금융보안원 및 KISA의 대응 공지 확인.

  • 공급망 공격: Checkmarx KICS의 공식 Docker 이미지 및 VS Code 확장 프로그램이 악성코드로 교체되어 Bitwarden CLI 침해로 이어진 사례 보고.


    • 4. 주요 CVE / 취약점


  • CVE-2026-0300: Palo Alto PAN-OS User-ID 인증 포털, Buffer Overflow 취약점, Active Exploitation 중.

  • CVE-2026-31431: Linux Kernel LPE(Copy Fail), 심각도 Critical, 루트 권한 획득 가능.

  • CVE-2024-52911: Bitcoin Core, 원격 코드 실행 가능, 43% 노드 미패치 상태.

  • CVE-2026-41940: cPanel & WHM, 인증 우회(Auth Bypass), PoC 공개됨.

  • CVE-2026-44331: ProFTPD, 보안 제한 우회(Security Restrictions Bypass), PoC 공개됨.
