1. 현재 주요 사이버보안 트렌드


최근 사이버 위협 환경은 LLM 에이전트를 활용한 공격 자동화가 실전 단계에 진입하며, 공격자가 인간의 개입 없이 엔드투엔드(End-to-End)로 침투를 수행하는 양상을 보입니다. 특히 공급망 공격에서 AI 환각(Hallucination)을 악용한 'Phantom Squatting' 기법이 부상하고 있으며, 취약점 패치 주기가 단축됨에 따라 방어 측의 신속한 대응 역량이 더욱 중요해졌습니다. 또한, 임베디드 기기 및 IoT 인프라의 취약점(FatFs 등)이 대규모 봇넷(Popa Botnet 등)의 숙주로 활용되는 사례가 지속적으로 보고되고 있습니다.

2. 주요 위협 행위자 / 캠페인


  • JadePuffer: LLM 에이전트를 활용하여 공격 전 과정을 자동화한 최초의 랜섬웨어 사례로 보고됨.

  • North Korean Hackers (Contagious Interview): npm, Packagist, Go, Chrome 확장 프로그램 등 108개의 악성 패키지를 배포하는 공급망 공격 수행.

  • Armored Likho (APT): 러시아 조직을 타겟으로 스피어 피싱 및 AI 생성 로더를 활용하며, 신규 Python 기반 BusySnake Stealer를 사용.

  • Scattered Spider: 최근 영국 Transport for London 공격 관련 혐의로 주요 조직원들이 유죄를 인정하거나 미국으로 인도됨.

  • TAG-182 (Iran-Nexus): MarkiRAT 감시 도구와 가짜 VPN/미디어 앱을 활용하여 국내 타겟 대상 사이버 감시 작전 수행.


    • 3. 한국 관련 위협


  • Settra: 한국 산업장비 제조 대기업의 해외 법인을 타겟으로 데이터 유출을 주장하는 신규 데이터 강탈 그룹 활동 포착.

  • Kimsuky: 보안 담당자를 사칭하여 개인정보 유출 문의 메일로 위장한 스피어 피싱 공격 지속.

  • 국내 APT 동향: 2026년 5월 기준, 국내 타겟 APT 공격이 지속적으로 모니터링되고 있으며, 웹/서버 애플리케이션 및 원격관리 도구 취약점 악용이 주를 이룸.


    • 4. 주요 CVE / 취약점


  • CVE-2026-46242: Linux Kernel (Bad Epoll), 로컬 권한 상승(Root), 악용 중.

  • CVE-2026-45504: Microsoft Exchange, SSRF 취약점, PoC 공개됨.

  • CVE-2026-6682/6687/6688: FatFs, CVSS 7.6, 임베디드 기기 타겟.

  • CVE-2026-26114: Microsoft SharePoint, 역직렬화 취약점, PoC 발견.

  • CVE-2025-3248: Langflow, Python 코드 실행, CISA KEV 등재.


    • 5. IOC 요약


  • 도메인: phantom-squatting 관련 도메인 (주의 필요) (Unit42)

  • 해시: chai-as-init (npm 패키지) (이스트시큐리티), BusySnake Stealer (샘플 확인 필요) (Securelist)

  • CVE: CVE-2026-46242 (TheHackerNews), CVE-2026-45504 (Undercode)


    • 6. 즉각 대응 권고


  • 공급망 보안 강화: npm, Go 등 오픈소스 패키지 사용 시 의존성 패키지의 출처를 검증하고, 특히 chai-as-init 등 의심스러운 패키지 사용 여부를 즉시 감사하십시오.

  • 임베디드/IoT 패치: FatFs 라이브러리를 사용하는 임베디드 기기(카메라, 드론, 산업 제어기)의 펌웨어 업데이트 여부를 확인하고, USB/SD 카드 인터페이스에 대한 물리적 보안 정책을 강화하십시오.

  • AI 에이전트 모니터링: Langflow 등 AI 자동화 도구를 운영 중인 경우, 최신 버전으로 패치하고 외부 접근이 가능한 엔드포인트에 대한 인증을 강화하십시오.
